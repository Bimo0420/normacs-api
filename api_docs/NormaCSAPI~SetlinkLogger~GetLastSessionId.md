# GetLastSessionId

## Описание

Возвращает идентификатор последней сессии расстановки ссылок

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetLastSessionId (Tag)
```

Метод GetLastSessionId состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| Tag | Обязательный. Значение String. |
