# ReadForAllUsers

## Описание

Заполняет коллекцию из хранилища для всех пользователей

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ReadForAllUsers**

Метод ReadForAllUsers состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
