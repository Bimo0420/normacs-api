# Domains (Recognize2Settings)

## Описание

Список доменов разыскиваемых документов, разделенных символом точки с запятой. Если этот параметр не указан, используются все подключенные домены.

## Тип свойства

Значение Variant.

## Синтаксис

*object***.Domains** [= *значение*]

Свойство Domains состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
| значение | Значение Variant. |
