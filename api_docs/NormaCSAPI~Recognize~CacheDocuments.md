# CacheDocuments

## Описание

Кешировать коллекции документов для распознанных обозначений. Используется для ускорения работы. По умолчанию установлено.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.CacheDocuments** [= *значение*]

Свойство CacheDocuments состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение Boolean. |
