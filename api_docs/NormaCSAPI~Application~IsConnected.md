# IsConnected

## Описание

Имеется ли соединение с базой данных. Соединение может быть выполнено явно при помощи метода Connect, или неявно, при вызове любого метода или свойства, запрашивающего данные из базы.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.IsConnected**

Свойство IsConnected состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
