# Item (UserFields)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [UserField](NormaCSAPI~UserField.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserFields](NormaCSAPI~UserFields.md). |
| Index | Обязательный. Значение Variant. |
