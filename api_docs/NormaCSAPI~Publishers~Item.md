# Item (Publishers)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Publisher](NormaCSAPI~Publisher.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Publishers](NormaCSAPI~Publishers.md). |
| Index | Обязательный. Значение Variant. |
