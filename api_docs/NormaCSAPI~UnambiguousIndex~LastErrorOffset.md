# LastErrorOffset (UnambiguousIndex)

## Описание

Возвращает позицию места ошибки в регулярном выражении, начиная с нуля

## Тип свойства

Значение Integer.

## Синтаксис

*object***.LastErrorOffset**

Свойство LastErrorOffset состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndex](NormaCSAPI~UnambiguousIndex.md). |
