# ReadForCurrentUserOrForAllUsers

## Описание

Заполняет коллекцию из хранилища для текущего пользователя или, если последнее не содержит элементов, из хранилища для всех пользователей

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ReadForCurrentUserOrForAllUsers**

Метод ReadForCurrentUserOrForAllUsers состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
