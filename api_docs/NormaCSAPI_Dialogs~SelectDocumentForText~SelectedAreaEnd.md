# SelectedAreaEnd

Конечная позиция выделенной части текста начиная с единицы

## Тип свойства

Значение Integer.

## Синтаксис

*object***.SelectedAreaEnd**

Свойство SelectedAreaEnd состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SelectDocumentForText](NormaCSAPI_Dialogs~SelectDocumentForText.md). |
