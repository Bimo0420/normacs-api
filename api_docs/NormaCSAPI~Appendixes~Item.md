# Item (Appendixes)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Appendix](NormaCSAPI~Appendix.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Appendixes](NormaCSAPI~Appendixes.md). |
| Index | Обязательный. Значение Variant. |
