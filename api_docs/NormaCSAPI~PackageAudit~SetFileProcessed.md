# SetFileProcessed (PackageAudit)

## Описание

Сообщить, что аудит указанного файла выполнен с указанным статусом

## Возвращаемое значение

Нет.

## Синтаксис

*object***.SetFileProcessed** *Path*, *Status*

Метод SetFileProcessed состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [PackageAudit](NormaCSAPI~PackageAudit.md). |
| Path | Обязательный. Значение String. |
| Status | Обязательный. Значение Integer. |
