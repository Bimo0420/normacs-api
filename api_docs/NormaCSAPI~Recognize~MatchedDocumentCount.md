# MatchedDocumentCount

## Описание

Количество документов, обозначения которых совпадают с распознанным с точностью до разделителей

## Тип свойства

Значение Integer.

## Синтаксис

*object***.MatchedDocumentCount**

Свойство MatchedDocumentCount состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
