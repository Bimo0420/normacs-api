# Load

## Описание

Загружает настройки из файла по указанному пути

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Load** *bstrPath*

Метод Load состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
| bstrPath | Обязательный. Значение String. |
