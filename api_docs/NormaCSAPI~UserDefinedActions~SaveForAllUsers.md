# SaveForAllUsers

## Описание

Сохраняет коллекцию в хранилище для всех пользователей

## Возвращаемое значение

Нет.

## Синтаксис

*object***.SaveForAllUsers**

Метод SaveForAllUsers состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
