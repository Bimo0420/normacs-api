# AddDocumentRef

## Описание

Добавляет указанный документ к списку связей. Вызовите SaveDocument для сохранения в базе измененных атрибутов документа.

## Возвращаемое значение

Объект [Ref](NormaCSAPI~Ref.md).

## Синтаксис

```
object.AddDocumentRef (pDocument, [RefType], [RefDocument], [HRef], [Index], [Number], [Name], [Comment], [IdRef10])
```

Метод AddDocumentRef состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| pDocument | Обязательный. Объект [Document](NormaCSAPI~Document.md). |
| RefType | Необязательный. Значение Variant. |
| RefDocument | Необязательный. Значение Variant. |
| HRef | Необязательный. Значение Variant. |
| Index | Необязательный. Значение Variant. |
| Number | Необязательный. Значение Variant. |
| Name | Необязательный. Значение Variant. |
| Comment | Необязательный. Значение Variant. |
| IdRef10 | Необязательный. Значение Variant. |
