# SaveForCurrentUser

## Описание

Сохраняет коллекцию в хранилище для текущего пользователя

## Возвращаемое значение

Нет.

## Синтаксис

*object***.SaveForCurrentUser**

Метод SaveForCurrentUser состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
