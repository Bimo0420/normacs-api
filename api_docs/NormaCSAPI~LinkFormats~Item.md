# Item (LinkFormats)

## Описание

Возвращает формат по указанному индексу, имени или ссылке

## Тип свойства

Объект [LinkFormat](NormaCSAPI~LinkFormat.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [LinkFormats](NormaCSAPI~LinkFormats.md). |
| Index | Обязательный. Значение Variant. |
