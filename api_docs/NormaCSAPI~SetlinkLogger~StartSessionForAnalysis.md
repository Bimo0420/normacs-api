# StartSessionForAnalysis

## Описание

Начинает сессию анализа ссылок. Возвращает идентификатор сессии.

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.StartSessionForAnalysis (Tag, Path, Settings)
```

Метод StartSessionForAnalysis состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| Tag | Обязательный. Значение String. |
| Path | Обязательный. Значение String. |
| Settings | Обязательный. Объект Object. |
