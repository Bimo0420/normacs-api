# UseAddinConfiguration

## Описание

Использовать для распознавания данные конфигурации надстроек NormaCS для Microsoft Office. В этом случае параметр RegExp не используется. По умолчанию установлено.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.UseAddinConfiguration** [= *значение*]

Свойство UseAddinConfiguration состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение Boolean. |
