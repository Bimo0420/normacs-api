# DaysAfterLastUpdate

## Описание

Возвращает количество дней, прошедших со времени последнего обновления домена.Используйте эту функцию для напоминания пользователю о необходимости обновить базу данных.

## Тип свойства

Значение Long.

## Синтаксис

*object***.DaysAfterLastUpdate**

Свойство DaysAfterLastUpdate состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domain](NormaCSAPI~Domain.md). |
