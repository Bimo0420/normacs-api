# SearchSubfolders

## Описание

Если указан список разделов для поиска, то значение True означает, что будут найдены документы, входящие в указанные разделы, а также в дочерние разделы произвольной вложенности.Значение False означает, что будут найдены только документы из указанных разделов.По умолчанию SearchSubfolders = True.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.SearchSubfolders** [= *значение*]

Свойство SearchSubfolders состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| значение | Значение Boolean. |
