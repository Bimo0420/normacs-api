## Описание

Объект UserDefinedAction

## Методы и свойства

| Свойства |  | Свойства |
| --- | --- | --- |
|  | Свойства |
| [Command](NormaCSAPI~UserDefinedAction~Command.md) |  | Команда |
| [Description](NormaCSAPI~UserDefinedAction~Description.md) |  | Пояснмтельный текст |
| [IconFilePath](NormaCSAPI~UserDefinedAction~IconFilePath.md) |  | Путь на файл с иображением кнопки |

| [IsHidden](NormaCSAPI~UserDefinedAction~IsHidden.md) |  | Не показывать на панели инструментов |
| [Name](NormaCSAPI~UserDefinedAction~Name.md) |  | Название |
| [Seq](NormaCSAPI~UserDefinedAction~Seq.md) |  | Число, большее 100, определяющее относительное расположение кнопки на панели инструментов. Элемент с большим значением Seq располагается левее элемента с меньшим значением |
