# Document (Ref)

## Описание

Связанный документ. Пустое значение, если документ отсутствует в NormaCS или не предусматривается связью

## Тип свойства

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

**[Set]** *object***.Document** [= *Document*]

Свойство Document состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Ref](NormaCSAPI~Ref.md). |
| Document | Объект [Document](NormaCSAPI~Document.md). |
