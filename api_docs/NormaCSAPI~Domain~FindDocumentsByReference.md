# FindDocumentsByReference

## Возвращаемое значение

Объект [Documents](NormaCSAPI~Documents.md).

## Синтаксис

```
object.FindDocumentsByReference (RefType, HRef)
```

Метод FindDocumentsByReference состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domain](NormaCSAPI~Domain.md). |
| RefType | Обязательный. Значение Integer. |
| HRef | Обязательный. Значение String. |
