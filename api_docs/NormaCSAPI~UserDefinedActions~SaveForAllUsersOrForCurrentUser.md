# SaveForAllUsersOrForCurrentUser

## Описание

Сохраняет коллекцию в хранилище для всех пользователей, или, при неудаче, для текущего пользователя

## Возвращаемое значение

Нет.

## Синтаксис

*object***.SaveForAllUsersOrForCurrentUser**

Метод SaveForAllUsersOrForCurrentUser состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
