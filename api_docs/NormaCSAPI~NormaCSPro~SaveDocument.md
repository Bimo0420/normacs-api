# SaveDocument

## Описание

Сохраняет реквизиты документа

## Возвращаемое значение

Нет.

## Синтаксис

*object***.SaveDocument** *pDocument*

Метод SaveDocument состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| pDocument | Обязательный. Объект [Document](NormaCSAPI~Document.md). |
