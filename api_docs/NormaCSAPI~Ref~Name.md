# Name (Ref)

## Описание

Название документа. Указывается, если документ отсутствует в NormaCS.

## Тип свойства

Значение String.

## Синтаксис

*object***.Name** [= *значение*]

Свойство Name состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Ref](NormaCSAPI~Ref.md). |
| значение | Значение String. |
