# SerialNumber

## Описание

Возвращает серийный номер используемой версии NormaCS

## Тип свойства

Значение String.

## Синтаксис

*object***.SerialNumber**

Свойство SerialNumber состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
