# Respondents

## Описание

Возвращает список документов, на которые ссылается этот документ (респонденты)

## Тип свойства

Объект [Documents](NormaCSAPI~Documents.md).

## Синтаксис

*object***.Respondents**

Свойство Respondents состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
