# Item (Replacements)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Replacement](NormaCSAPI~Replacement.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Replacements](NormaCSAPI~Replacements.md). |
| Index | Обязательный. Значение Variant. |
