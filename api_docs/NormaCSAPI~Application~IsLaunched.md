# IsLaunched

## Описание

True если приложение NormaCS запущено

## Возвращаемое значение

Значение Boolean.

## Синтаксис

*object***.IsLaunched**

Метод IsLaunched состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
