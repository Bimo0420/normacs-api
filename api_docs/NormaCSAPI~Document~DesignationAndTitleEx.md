# DesignationAndTitleEx

## Описание

Возвращает обозначение и наименование документа с указанием синонимов и даты утверждения, если требуется.

## Тип свойства

Значение String.

## Синтаксис

```
object.DesignationAndTitleEx(NeedApproveDate)
```

Свойство DesignationAndTitleEx состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
| NeedApproveDate | Обязательный. Значение Boolean. |
