# NormaCSSetlinkLogReason

## Описание

Как была выполнена установка ссылки или почему ссылка была пропущена

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsSLLReasonSetInteractive | 1 | Ссылка установлена в диалоге |
|  | ncsSLLReasonSetAutoset | 2 | Ссылка установлена автоматически |
|  | ncsSLLReasonSetAutosetByRule | 3 | Ссылка установлена автоматически по образцу |
|  | ncsSLLReasonSkipNotReliableIndex | 10 | Установка ссылки не выполнена ввиду того, что распознанное обозначение имеет требующий-уточнения индекс |
|  | ncsSLLReasonSkipMatchedMoreThanOneDocument | 11 | Установка ссылки не выполнена ввиду того, что для распознанного обозначения имеется более чем один документ |
|  | ncsSLLReasonSkipObsoleted | 12 | Установка ссылки не выполнена ввиду того, что выбранный документ устарел |
|  | ncsSLLReasonSkipOverride | 13 | Установка ссылки не выполнена ввиду того, что на распознанный фрагмент уже установлена ссылка |
|  | ncsSLLReasonSkipNoContent | 14 | Установка ссылки не выполнена ввиду того, что выбранный документ документ не имеет ни текста, ни изображения |
|  | ncsSLLReasonSkipUserChoise | 15 | Установка ссылки отклонена пользователем |
|  | ncsSLLReasonSkipNoDocuments | 20 | Установка ссылки не выполнена ввиду того, что для распознанного обозначения не найдено ни одного документа |
|  | ncsSLLReasonBlockedProtectedArea | 35 |  |
|  | ncsSLLReasonBlockedUnknownReason | 36 |  |
|  | ncsSLLReasonSkipUnknown | 100 | Установка ссылки не выполнена по неизвестной причине |
