# ProblemItemsPositions

## Описание

Позиции проблемных элементов списка в исходном документе

## Тип свойства

Массив значений String

## Синтаксис

```
object.ProblemItemsPositions(FilteredOnly)
```

Свойство ProblemItemsPositions состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [VerifyLinks](NormaCSAPI_Dialogs~VerifyLinks.md). |
| FilteredOnly | Обязательный. Значение Boolean. |
