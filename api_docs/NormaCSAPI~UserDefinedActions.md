# UserDefinedActions Коллекция

## Описание

Объект UserDefinedActions

## Объектная модель

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [Add](NormaCSAPI~UserDefinedActions~Add.md) | Добавляет указанный элемент (документ) в коллекцию |
| [Clear](NormaCSAPI~UserDefinedActions~Clear.md) | Удаляет все элементы из коллекции |
| [CreateAction](NormaCSAPI~UserDefinedActions~CreateAction.md) | Создает объект надстройки |
| [ReadForAllUsers](NormaCSAPI~UserDefinedActions~ReadForAllUsers.md) | Заполняет коллекцию из хранилища для всех пользователей |
| [ReadForCurrentUser](NormaCSAPI~UserDefinedActions~ReadForCurrentUser.md) | Заполняет коллекцию из хранилища для текущего пользователя |
| [ReadForCurrentUserOrForAllUsers](NormaCSAPI~UserDefinedActions~ReadForCurrentUserOrForAllUsers.md) | Заполняет коллекцию из хранилища для текущего пользователя или, если последнее не содержит элементов, из хранилища для всех пользователей |
| [Remove](NormaCSAPI~UserDefinedActions~Remove.md) | Удаляет элемент (документ) с указанным индексом из коллекции |
| [SaveForAllUsers](NormaCSAPI~UserDefinedActions~SaveForAllUsers.md) | Сохраняет коллекцию в хранилище для всех пользователей |
| [SaveForAllUsersOrForCurrentUser](NormaCSAPI~UserDefinedActions~SaveForAllUsersOrForCurrentUser.md) | Сохраняет коллекцию в хранилище для всех пользователей, или, при неудаче, для текущего пользователя |
| [SaveForCurrentUser](NormaCSAPI~UserDefinedActions~SaveForCurrentUser.md) | Сохраняет коллекцию в хранилище для текущего пользователя |
