# Restrictions

## Описание

Ограничения на использование документа. Значением может быть: 0(нет ограничений); 1(экспорт запрещен); 3(экспорт и печать запрещены); 7(экспорт, печать и цитирование запрещены)

## Тип свойства

Значение Long.

## Синтаксис

*object***.Restrictions** [= *значение*]

Свойство Restrictions состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
| значение | Значение Long. |
