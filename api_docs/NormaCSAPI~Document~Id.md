# Id (Document)

## Описание

Возвращает 32-ричный идентификатор документа внутри домена, к которому принадлежит данный документ. Вместе с доменом однозначно определяет документ в системе NormaCS

## Тип свойства

Значение String.

## Синтаксис

*object***.Id**

Свойство Id состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
