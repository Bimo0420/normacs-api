# Designation (Document)

## Описание

Возвращает обозначение документа, построенное путем конкатенации основного индекса, разделителя и основного номера

## Тип свойства

Значение String.

## Синтаксис

*object***.Designation**

Свойство Designation состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
