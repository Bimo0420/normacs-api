# RemovePublisher

## Описание

Удаляет указанную организацию из справочника издающих организаций домена

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemovePublisher** *Publisher*

Метод RemovePublisher состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Publisher | Обязательный. Значение Variant. |
