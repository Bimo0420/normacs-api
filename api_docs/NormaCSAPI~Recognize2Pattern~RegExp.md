# RegExp (Recognize2Pattern)

## Описание

Задает регулярное выражение в формате PCRE, с группами захвата для индекса и номера распознаваемого обозначения.

## Тип свойства

Значение String.

## Синтаксис

*object***.RegExp** [= *значение*]

Свойство RegExp состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Pattern](NormaCSAPI~Recognize2Pattern.md). |
| значение | Значение String. |
