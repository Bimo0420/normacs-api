# Remove (UserDefinedActions)

## Описание

Удаляет элемент (документ) с указанным индексом из коллекции

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Remove** *Index*

Метод Remove состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
| Index | Обязательный. Значение Variant. |
