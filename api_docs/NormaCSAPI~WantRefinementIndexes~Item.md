# Item (WantRefinementIndexes)

## Описание

Возвращает шаблон распознавания по указанному индексу, имени или ссылке

## Тип свойства

Объект [WantRefinementIndex](NormaCSAPI~WantRefinementIndex.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [WantRefinementIndexes](NormaCSAPI~WantRefinementIndexes.md). |
| Index | Обязательный. Значение Variant. |
