# GetRevisionByURL

## Описание

Возвращает редакцию с указанным адресом

## Возвращаемое значение

Объект [Revision](NormaCSAPI~Revision.md).

## Синтаксис

```
object.GetRevisionByURL (URL)
```

Метод GetRevisionByURL состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| URL | Обязательный. Значение String. |
