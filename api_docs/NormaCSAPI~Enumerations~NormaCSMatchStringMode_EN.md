# NormaCSMatchStringMode

## Описание

Критерий успеха при поиске по строке

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsMatchAnyPartOfString | 0 | Совпадение образца с любой частью строки |
|  | ncsMatchWholeString | 1 | Совпадение образца со всей строкой |
|  | ncsMatchFromBeginningOfString | 2 | Совпадение образца с началом строки |
