# ExtraDesignations

## Описание

Возвращает CSV-список дополнительных обозначений документа, то есть, список синонимов исключая основной

## Тип свойства

Значение String.

## Синтаксис

*object***.ExtraDesignations**

Свойство ExtraDesignations состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
