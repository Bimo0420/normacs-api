# CurrentPage

## Описание

Возвращает текущую страницу изображения документа или ноль. Нумерация страниц начинается с единицы. Данное свойство имеет смысл только в случае, если документ открыт в NormaCS как изображение и нажата кнопка [Копировать ссылку на текущий документ]

## Тип свойства

Значение Integer.

## Синтаксис

*object***.CurrentPage**

Свойство CurrentPage состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
