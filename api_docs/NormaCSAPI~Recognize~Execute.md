# Execute (Recognize)

## Описание

Выполняет распознавание обозначения документа в тексте. При успешном распознавании заполняет список документов, подходящих к распознанному обозначению и записывает в NextPos позицию для следующего вызова.

## Возвращаемое значение

Значение Boolean.

## Синтаксис

*object***.Execute**

Метод Execute состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
