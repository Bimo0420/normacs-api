# SelectedIndex

## Описание

Индекс активного формата, начинается с единицы

## Тип свойства

Значение Integer.

## Синтаксис

*object***.SelectedIndex** [= *значение*]

Свойство SelectedIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [LinkFormats](NormaCSAPI~LinkFormats.md). |
| значение | Значение Integer. |
