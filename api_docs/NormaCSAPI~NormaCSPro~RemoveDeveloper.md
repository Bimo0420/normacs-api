# RemoveDeveloper

## Описание

Удаляет указанную организацию из справочника разрабатывающих организаций домена

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveDeveloper** *Developer*

Метод RemoveDeveloper состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Developer | Обязательный. Значение Variant. |
