# AddPublisher

## Описание

Добавляет указанную организацию к справочнику издающих организаций домена

## Возвращаемое значение

Объект [Publisher](NormaCSAPI~Publisher.md).

## Синтаксис

```
object.AddPublisher (Name)
```

Метод AddPublisher состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Name | Обязательный. Значение String. |
