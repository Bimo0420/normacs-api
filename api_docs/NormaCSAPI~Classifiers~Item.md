# Item (Classifiers)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Classifier](NormaCSAPI~Classifier.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Classifiers](NormaCSAPI~Classifiers.md). |
| Index | Обязательный. Значение Variant. |
