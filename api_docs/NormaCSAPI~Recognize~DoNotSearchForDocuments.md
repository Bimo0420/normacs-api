# DoNotSearchForDocuments

## Описание

Не выполнять поиск документов для распознанного обозначения. В этом случае коллекция Documents не заполняется. По умолчанию не установлено.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.DoNotSearchForDocuments** [= *значение*]

Свойство DoNotSearchForDocuments состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение Boolean. |
