# Approver (Find)

## Описание

Подстрока названия утверждающей организации для разыскиваемых документов. Может включать групповой символ (*).

## Тип свойства

Значение String.

## Синтаксис

*object***.Approver** [= *значение*]

Свойство Approver состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| значение | Значение String. |
