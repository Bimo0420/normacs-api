# Remove (Recognize2Patterns)

## Описание

Удаляет шаблон распознавания из списка шаблонов

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Remove** *Index*

Метод Remove состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Patterns](NormaCSAPI~Recognize2Patterns.md). |
| Index | Обязательный. Значение Variant. |
