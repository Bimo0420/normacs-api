# UnambiguousIndexes Коллекция

## Описание

Список шаблонов достоверных индексов

## Объектная модель

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [Add](NormaCSAPI~UnambiguousIndexes~Add.md) | Добавляет шаблон к списку шаблонов. Входноq параметр может быть индексом built-in-шаблона, именем создаваемого custom-шаблона, или ссылкой на существующий шаблон. |
| [AssignTo](NormaCSAPI~UnambiguousIndexes~AssignTo.md) | Копирует список шаблонов в указанный |
| [Has](NormaCSAPI~UnambiguousIndexes~Has.md) | Имеется ли в списке шаблонов шаблон с указанным именем или built-in индексом |
| [HasActive](NormaCSAPI~UnambiguousIndexes~HasActive.md) | Имеется ли в списке шаблонов хотя бы один активный шаблон |
| [IsMatched](NormaCSAPI~UnambiguousIndexes~IsMatched.md) | Является ли указанный индекс неоднозначным |
| [Normalize](NormaCSAPI~UnambiguousIndexes~Normalize.md) | Добавляет встроенные шаблоны распознавания к списку шаблонов |
| [Remove](NormaCSAPI~UnambiguousIndexes~Remove.md) | Удаляет шаблон распознавания из списка шаблонов |
| [RemoveAll](NormaCSAPI~UnambiguousIndexes~RemoveAll.md) | Удаляет все шаблоны распознавания из списка шаблонов |
| [Test](NormaCSAPI~UnambiguousIndexes~Test.md) |  |
