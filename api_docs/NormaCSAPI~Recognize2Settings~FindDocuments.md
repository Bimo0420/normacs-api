# FindDocuments

## Описание

Выполнять поиск документов для распознанного обозначения. По умолчанию установлено.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.FindDocuments** [= *значение*]

Свойство FindDocuments состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
| значение | Значение Boolean. |
