# Item (UnambiguousIndexes)

## Описание

Возвращает шаблон распознавания по указанному индексу, имени или ссылке

## Тип свойства

Объект [UnambiguousIndex](NormaCSAPI~UnambiguousIndex.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
| Index | Обязательный. Значение Variant. |
