# RegExp (UnambiguousIndexes)

## Описание

Возвращает регулярное выражение для разбора, представляющее собой или-конкатенацию развернутых регулярных выражений активных шаблонов списка

## Тип свойства

Значение String.

## Синтаксис

*object***.RegExp**

Свойство RegExp состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
