# FindIndex

## Описание

Возвращает индекс документа для указанного обозначения документа или Null, если индекс не удается определить

## Возвращаемое значение

Объект [Index](NormaCSAPI~Index.md).

## Синтаксис

```
object.FindIndex (Designation)
```

Метод FindIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domain](NormaCSAPI~Domain.md). |
| Designation | Обязательный. Значение String. |
