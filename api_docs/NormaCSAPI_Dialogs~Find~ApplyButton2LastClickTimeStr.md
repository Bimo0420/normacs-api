# ApplyButton2LastClickTimeStr

## Описание

Строка, содержащая время последнего вызова пункта меню [Установить на текст] в формате YYYY-MM-DD HH:MM:SS. Может быть использована для отслеживания вызовов без помощи обработчика событий.

## Тип свойства

Значение String.

## Синтаксис

*object***.ApplyButton2LastClickTimeStr**

Свойство ApplyButton2LastClickTimeStr состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
