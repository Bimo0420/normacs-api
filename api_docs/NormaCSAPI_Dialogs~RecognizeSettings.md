# RecognizeSettings Объект

## Описание

Диалог настроек распознавания обозначений документов

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [ShowModal](NormaCSAPI_Dialogs~RecognizeSettings~ShowModal.md) | Показывает модальный диалог для редактирования настроек распознавания |
| [ShowModal2](NormaCSAPI_Dialogs~RecognizeSettings~ShowModal2.md) | Показывает модальный диалог для редактирования настроек распознавания. В отличие от ShowModal, позволяет указать дескриптор родительского окна. |
