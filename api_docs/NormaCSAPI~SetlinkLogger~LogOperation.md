# LogOperation

## Описание

Записывает в протокол расстановки ссылок

## Возвращаемое значение

Нет.

## Синтаксис

*object***.LogOperation** *SessionId*, *Operation*, *Reason*, *Text*, *StartPos*, *EndPos*, *Location*, *DocumentUrl*, *DocumentName*, *documentIconIndex*

Метод LogOperation состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| SessionId | Обязательный. Значение String. |
| Operation | Обязательный. Перечисление [NormaCSSetlinkLogOperation](NormaCSAPI~Enumerations~NormaCSSetlinkLogOperation_EN.md). |
| Reason | Обязательный. Перечисление [NormaCSSetlinkLogReason](NormaCSAPI~Enumerations~NormaCSSetlinkLogReason_EN.md). |
| Text | Обязательный. Значение String. |
| StartPos | Обязательный. Значение Integer. |
| EndPos | Обязательный. Значение Integer. |
| Location | Обязательный. Значение String. |
| DocumentUrl | Обязательный. Значение String. |
| DocumentName | Обязательный. Значение String. |
| documentIconIndex | Обязательный. Значение Integer. |
