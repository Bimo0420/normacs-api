# NormaCSDocumentTextFormat

## Описание

Формат источника текстового представления документа

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsDocumentTextFormatUnknown | 0 | Формат недоступен или нераспознан |
|  | ncsDocumentTextFormatHtml | 1 | Формат HTML |
|  | ncsDocumentTextFormatDoc | 2 | Формат MSWord DOC или DOCX |
|  | ncsDocumentTextFormatRtf | 3 | Формат RTF |
