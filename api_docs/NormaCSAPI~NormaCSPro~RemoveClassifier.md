# RemoveClassifier

## Описание

Удаляет указанный классификатор

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveClassifier** *Classifier*

Метод RemoveClassifier состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Classifier | Обязательный. Значение Variant. |
