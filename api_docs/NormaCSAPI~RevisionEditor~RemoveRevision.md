# RemoveRevision

## Описание

Удаляет редакцию. Редакция не должна быть основной.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveRevision** *Revision*

Метод RemoveRevision состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [RevisionEditor](NormaCSAPI~RevisionEditor.md). |
| Revision | Обязательный. Значение Variant. Сслылка на редакцию, или URL редакции. |
