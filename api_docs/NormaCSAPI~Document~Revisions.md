# Revisions

## Описание

Коллекция редакций документа. Используйте объект [RevisionEditor](NormaCSAPI~NormaCSPro~RevisionEditor.md) для создания, изменения и удаления редакций.

## Тип свойства

Объект [Revisions](NormaCSAPI~Revisions.md).

## Синтаксис

*object***.Revisions**

Свойство Revisions состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
