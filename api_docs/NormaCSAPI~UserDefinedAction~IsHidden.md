# IsHidden

## Описание

Не показывать на панели инструментов

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.IsHidden** [= *значение*]

Свойство IsHidden состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedAction](NormaCSAPI~UserDefinedAction.md). |
| значение | Значение Boolean. |
