# Has (Indexes)

## Описание

Есть ли в коллекции указанный элемент

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.Has (Name)
```

Метод Has состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Indexes](NormaCSAPI~Indexes.md). |
| Name | Обязательный. Значение String. |
