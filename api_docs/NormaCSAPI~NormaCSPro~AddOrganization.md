# AddOrganization

## Описание

Добавляет указанную организацию к справочнику организаций домена

## Возвращаемое значение

Объект [Organization](NormaCSAPI~Organization.md).

## Синтаксис

```
object.AddOrganization (Name)
```

Метод AddOrganization состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Name | Обязательный. Значение String. |
