# Item (Keywords)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Keyword](NormaCSAPI~Keyword.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Keywords](NormaCSAPI~Keywords.md). |
| Index | Обязательный. Значение Variant. |
