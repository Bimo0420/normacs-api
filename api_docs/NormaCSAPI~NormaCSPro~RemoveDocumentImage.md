# RemoveDocumentImage

## Описание

Удаляет изображение документа

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveDocumentImage** *Document*

Метод RemoveDocumentImage состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Document | Обязательный. Значение Variant. |
