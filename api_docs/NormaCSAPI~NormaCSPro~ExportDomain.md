# ExportDomain

## Описание

Формирует пакет с базой и документами для сторонней установки домена

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ExportDomain** *OutputFolder*, [*DontExportDocumentFiles*], [*ExportDocumentFilesNotOlderThen*], [*PlaceDocumentFilesInsidePack*], [*KeepExistedDocumentFiles*], [*IgnoreDocumentErrors*], [*LogFilePath*]

Метод ExportDomain состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| OutputFolder | Обязательный. Значение Variant. Путь на папку для размещения формируемого пакета и документов. |
| DontExportDocumentFiles | Необязательный. Значение Variant. Не добавлять файлы документов в папку экспорта ( при необходимости, файлы документов можно будет вручную скопировать из хранилища). По умолчанию файлы документов добавляются. |
| ExportDocumentFilesNotOlderThen | Необязательный. Значение Variant. Добавить только файлы документов, внесенные в базу начиная с указанной даты. По умолчанию добавляются все файлы документов. |
| PlaceDocumentFilesInsidePack | Необязательный. Значение Variant. Поместить файлы документов в пакет обновления. По умолчанию файлы документов размещаются отдельно от пакета в папке <OutputFolder>\addonBase\<Domain>. |
| KeepExistedDocumentFiles | Необязательный. Значение Variant. Для ускорения сборки использовать ранее сформированные для сборки файлы документов в папке addonBase\\<домен>. По умолчанию ранее сформированные файлы не сохраняются, указанная папка пересоздается при каждой сборке, требующей экспорт файлов документов.Этот параметр не используется, если указан флаг PlaceDocumentFilesInsidePack.Не влияет на результат сборки. |
| IgnoreDocumentErrors | Необязательный. Значение Variant. Не прерывать обработку при ошибках, связанных с отдельными документами.Обратите внимание, что при при выборе этой опции при установке пакета у пользователя возможны ошибки, связанные с отсутствием всех или некоторых файлов проблемных документов. |
| LogFilePath | Необязательный. Значение Variant. Помещать лог экспорта по указанному пути. По умолчанию лог создается в файле ProgramData\\Nanosoft\\NormaCS\\log\\npu_*.log. |
