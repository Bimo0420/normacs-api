# Item (UserDefinedActions)

## Описание

Возвращает элемент коллекции (документ) по указанному индексу

## Тип свойства

Объект [UserDefinedAction](NormaCSAPI~UserDefinedAction.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
| Index | Обязательный. Значение Variant. |
