# Topmost

## Описание

Поместить диалог поверх других окон

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.Topmost** [= *значение*]

Свойство Topmost состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| значение | Значение Boolean. |
