# GetLogAsHtml

## Описание

Возвращает html-представление протокола указанной сессии расстановки ссылок

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetLogAsHtml (SessionId)
```

Метод GetLogAsHtml состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| SessionId | Обязательный. Значение String. |
