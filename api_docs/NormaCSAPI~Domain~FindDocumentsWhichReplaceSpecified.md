# FindDocumentsWhichReplaceSpecified

## Описание

Возвращает список документов, заменяющих указанный

## Возвращаемое значение

Объект [Documents](NormaCSAPI~Documents.md).

## Синтаксис

```
object.FindDocumentsWhichReplaceSpecified (Designation)
```

Метод FindDocumentsWhichReplaceSpecified состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domain](NormaCSAPI~Domain.md). |
| Designation | Обязательный. Значение String. |
