# URLEx

## Описание

Как URL, но с добавлением текущей даты и версии домена. Указанная информация используется для контроля изменения статуса документа с момента добавления ссылки.

## Тип свойства

Значение String.

## Синтаксис

*object***.URLEx**

Свойство URLEx состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
