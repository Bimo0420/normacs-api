# GetDocumentByDomainId10

## Описание

Возвращает документ с указанным доменом и идентификатором в десятичном формате

## Возвращаемое значение

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

```
object.GetDocumentByDomainId10 (Domain, Id10)
```

Метод GetDocumentByDomainId10 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| Domain | Обязательный. Значение String. |
| Id10 | Обязательный. Значение String. |
