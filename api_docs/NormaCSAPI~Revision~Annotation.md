# Annotation

## Описание

Аннотация к редакции. Для изменения используйте метод NormaCSPro.RevisionEditor.UpdateRevision.

## Тип свойства

Значение String.

## Синтаксис

*object***.Annotation**

Свойство Annotation состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Revision](NormaCSAPI~Revision.md). |

## Замечания

В списке редакций приложения NormaCS Pro этому атрибуту соответствует поле "Аннотация".
