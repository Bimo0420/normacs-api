# CurrentDocument

## Описание

Возвращает текущий документ NormaCS. Предварительно следует запустить NormaCS и выбрать нужный документ в списке навигатора или среди открытых вкладок.

## Тип свойства

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

*object***.CurrentDocument**

Свойство CurrentDocument состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
