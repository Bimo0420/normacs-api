# ShowModal (SetlinkReportSummary)

## Описание

Показывает модальный диалог для указанной сессии расстановки ссылок

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.ShowModal (SessionId, [HwndParent])
```

Метод ShowModal состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkReportSummary](NormaCSAPI_Dialogs~SetlinkReportSummary.md). |
| SessionId | Обязательный. Значение String. |
| HwndParent | Необязательный. Значение Variant. |
