# FindByTextMode

## Описание

Модификатор поиска по тексту документа

## Тип свойства

Перечисление [NormaCSFindByTextMode](NormaCSAPI~Enumerations~NormaCSFindByTextMode_EN.md).

## Синтаксис

*object***.FindByTextMode** [= *значение*]

Свойство FindByTextMode состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| значение | Перечисление [NormaCSFindByTextMode](NormaCSAPI~Enumerations~NormaCSFindByTextMode_EN.md). |

## Настройки

Настройки для *значение*:

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsFindByTextAnyWord | 0 | Искать любую комбинацию из указанных слов (установлено по умолчанию) |
|  | ncsFindByTextAllWordsInAnyOrder | 1 | Искать все указанные слова в любом порядке |
|  | ncsFindByTextAllWordsInSpecifiedOrder | 2 | Искать все указанные слова в указанном порядке |
