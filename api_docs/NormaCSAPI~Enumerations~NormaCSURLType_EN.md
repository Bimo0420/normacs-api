# NormaCSURLType

## Описание

Тип объекта NormaCS, на который указывает URL

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsURLTypeDocument | 0 | URL указывает на документ |
|  | ncsURLTypeFolder | 1 | URL указывает на папку |
|  | ncsURLTypeAmendment | 2 | URL указывает на изменение к документу |
|  | ncsURLTypeUnknown | 100 | URL не распознано |
