# CreateDocuments

## Описание

Возвращает пустой список документов для дальнейшего его заполнения пользователем

## Возвращаемое значение

Объект [Documents](NormaCSAPI~Documents.md).

## Синтаксис

*object***.CreateDocuments**

Метод CreateDocuments состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
