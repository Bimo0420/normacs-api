# HasLog

## Описание

Возвращает True если в базе имеются данные о расстановке ссылок с указанным тэгом

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.HasLog (Tag)
```

Метод HasLog состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| Tag | Обязательный. Значение String. |
