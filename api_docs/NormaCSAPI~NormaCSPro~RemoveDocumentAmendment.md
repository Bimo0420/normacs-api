# RemoveDocumentAmendment

## Описание

Удаляет указанное изменение. Вызовите SaveDocument для сохранения в базе измененных атрибутов документа.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveDocumentAmendment** *pDocument*, *Amendment*

Метод RemoveDocumentAmendment состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| pDocument | Обязательный. Объект [Document](NormaCSAPI~Document.md). |
| Amendment | Обязательный. Значение Variant. |
