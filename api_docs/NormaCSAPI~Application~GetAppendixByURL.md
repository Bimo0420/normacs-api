# GetAppendixByURL

## Описание

Возвращает приложение с указанным адресом

## Возвращаемое значение

Объект [Appendix](NormaCSAPI~Appendix.md).

## Синтаксис

```
object.GetAppendixByURL (URL)
```

Метод GetAppendixByURL состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| URL | Обязательный. Значение String. |
