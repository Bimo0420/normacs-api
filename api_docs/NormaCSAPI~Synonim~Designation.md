# Designation (Synonim)

## Описание

Возвращает обозначение документа, построенное из индекса и номера

## Тип свойства

Значение String.

## Синтаксис

*object***.Designation**

Свойство Designation состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Synonim](NormaCSAPI~Synonim.md). |
