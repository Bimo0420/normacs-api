# IconFilePath

## Описание

Путь на файл с иображением кнопки

## Тип свойства

Значение String.

## Синтаксис

*object***.IconFilePath** [= *значение*]

Свойство IconFilePath состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedAction](NormaCSAPI~UserDefinedAction.md). |
| значение | Значение String. |
