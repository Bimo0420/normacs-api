# Item (Documents)

## Описание

Возвращает элемент коллекции (документ) по указанному индексу

## Тип свойства

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Documents](NormaCSAPI~Documents.md). |
| Index | Обязательный. Значение Variant. |
