# IsRevision

## Описание

Является ли документ редакцией другого документа

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.IsRevision**

Свойство IsRevision состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
