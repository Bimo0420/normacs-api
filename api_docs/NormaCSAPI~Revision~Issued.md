# Issued (Revision)

## Описание

Дата ввода редакции в действие. Для изменения используйте метод NormaCSPro.RevisionEditor.UpdateRevision.

## Тип свойства

Значение Date.

## Синтаксис

*object***.Issued**

Свойство Issued состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Revision](NormaCSAPI~Revision.md). |

## Замечания

В списке редакций приложения NormaCS Pro этому атрибуту соответствует поле "От".
