# ShowModal (SelectDocumentForText)

## Описание

Показывает модальный диалог выбора документа для указанного текста

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.ShowModal ([HwndParent])
```

Метод ShowModal состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SelectDocumentForText](NormaCSAPI_Dialogs~SelectDocumentForText.md). |
| HwndParent | Необязательный. Значение Variant. |
