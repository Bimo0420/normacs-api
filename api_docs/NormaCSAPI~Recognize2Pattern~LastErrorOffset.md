# LastErrorOffset (Recognize2Pattern)

## Описание

Возвращает позицию места ошибки в регулярном выражении, начиная с нуля

## Тип свойства

Значение Integer.

## Синтаксис

*object***.LastErrorOffset**

Свойство LastErrorOffset состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Pattern](NormaCSAPI~Recognize2Pattern.md). |
