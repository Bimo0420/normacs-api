# DateFinish (Revision)

## Описание

Дата окончания действия редакции. Для изменения используйте метод NormaCSPro.RevisionEditor.UpdateRevision.

## Тип свойства

Значение Date.

## Синтаксис

*object***.DateFinish**

Свойство DateFinish состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Revision](NormaCSAPI~Revision.md). |

## Замечания

В списке редакций приложения NormaCS Pro этому атрибуту соответствует поле "Дата окончания".
