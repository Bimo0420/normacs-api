# Item (Organizations)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Organization](NormaCSAPI~Organization.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Organizations](NormaCSAPI~Organizations.md). |
| Index | Обязательный. Значение Variant. |
