# DocumentCountTotal

## Описание

Возвращает полное количество документов для домена, включая документы без текста и/или изображения, а также документы из неприобретенных разделов

## Тип свойства

Значение Long.

## Синтаксис

*object***.DocumentCountTotal**

Свойство DocumentCountTotal состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domain](NormaCSAPI~Domain.md). |
