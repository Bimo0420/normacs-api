# GetFormattedText (Amendment)

## Описание

Формирует текст по указанному формату из атрибутов изменения

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetFormattedText (Format)
```

Метод GetFormattedText состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Amendment](NormaCSAPI~Amendment.md). |
| Format | Обязательный. Значение String. |
