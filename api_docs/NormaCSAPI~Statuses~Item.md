# Item (Statuses)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Status](NormaCSAPI~Status.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Statuses](NormaCSAPI~Statuses.md). |
| Index | Обязательный. Значение Variant. |
