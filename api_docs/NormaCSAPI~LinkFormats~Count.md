# Count (LinkFormats)

## Описание

Возвращает количество элементов в списке форматов

## Тип свойства

Значение Long.

## Синтаксис

*object***.Count**

Свойство Count состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [LinkFormats](NormaCSAPI~LinkFormats.md). |
