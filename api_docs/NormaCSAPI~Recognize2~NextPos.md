# NextPos (Recognize2)

## Описание

Позиция в тексте с которой искать, начиная с единицы

## Тип свойства

Значение Integer.

## Синтаксис

*object***.NextPos** [= *значение*]

Свойство NextPos состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2](NormaCSAPI~Recognize2.md). |
| значение | Значение Integer. |
