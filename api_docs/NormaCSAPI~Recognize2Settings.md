# Recognize2Settings Объект

## Описание

Настройки для распознавания обозначений документов

## Объектная модель

## Методы и свойства

Искать документ с обозначеним, имеющим цифры года, соответствующие распознанному обозначению. По умолчанию не установлено

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [AssignTo](NormaCSAPI~Recognize2Settings~AssignTo.md) | Копирует содержимое в указанный объект настроек |
| [CreateNewPattern](NormaCSAPI~Recognize2Settings~CreateNewPattern.md) | Создает новый объект шаблона |
| [CreateNewSettings](NormaCSAPI~Recognize2Settings~CreateNewSettings.md) | Создает новый объект настроек |
| [CreateNewUnambiguousIndex](NormaCSAPI~Recognize2Settings~CreateNewUnambiguousIndex.md) | Создает новый шаблон достоверного индекса |
| [Load](NormaCSAPI~Recognize2Settings~Load.md) | Загружает настройки из файла по указанному пути |
| [ResetToDefaults](NormaCSAPI~Recognize2Settings~ResetToDefaults.md) | Устанавливает настройки по умолчанию |
| [Save](NormaCSAPI~Recognize2Settings~Save.md) | Сохраняет настройки в файле по указанному пути |
