# Connect

## Описание

Открывает соединение с подключенными базами данных. Вызывать необязательно, так как соединение автоматически открывается при вызове любого метода или свойства, запрашивающего данные из базы.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Connect**

Метод Connect состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
