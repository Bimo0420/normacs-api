# GetSummaryAsXml

## Описание

Возвращает xml-представление сводной информации для указанной сессии расстановки ссылок

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetSummaryAsXml (SessionId)
```

Метод GetSummaryAsXml состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| SessionId | Обязательный. Значение String. |
