# URL (Document)

## Описание

Возвращает URL документа, построенное из текущих значений домена, идентификатора и номера страницы изображения (если указан)

## Тип свойства

Значение String.

## Синтаксис

*object***.URL**

Свойство URL состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
