# Index (Ref)

## Описание

Индекс документа. Указывается, если документ отсутствует в NormaCS.

## Тип свойства

Значение String.

## Синтаксис

*object***.Index** [= *значение*]

Свойство Index состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Ref](NormaCSAPI~Ref.md). |
| значение | Значение String. |
