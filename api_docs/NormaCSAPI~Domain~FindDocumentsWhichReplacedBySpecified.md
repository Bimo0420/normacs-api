# FindDocumentsWhichReplacedBySpecified

## Описание

Возвращает список документов, заменяемых указанным

## Возвращаемое значение

Объект [Documents](NormaCSAPI~Documents.md).

## Синтаксис

```
object.FindDocumentsWhichReplacedBySpecified (Designation)
```

Метод FindDocumentsWhichReplacedBySpecified состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domain](NormaCSAPI~Domain.md). |
| Designation | Обязательный. Значение String. |
