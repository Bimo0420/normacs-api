# Has (Folders)

## Описание

Есть ли в коллекции указанный элемент

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.Has (Name)
```

Метод Has состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Folders](NormaCSAPI~Folders.md). |
| Name | Обязательный. Значение String. |
