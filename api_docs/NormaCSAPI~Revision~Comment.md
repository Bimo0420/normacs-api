# Comment (Revision)

## Описание

Комментарий к редакции. Для изменения используйте метод NormaCSPro.RevisionEditor.UpdateRevision.

## Тип свойства

Значение String.

## Синтаксис

*object***.Comment**

Свойство Comment состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Revision](NormaCSAPI~Revision.md). |

## Замечания

В списке редакций приложения NormaCS Pro этому атрибуту соответствует поле "Комментарий".
