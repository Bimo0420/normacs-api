# CardXML

## Описание

XML-текст карточки документа

## Тип свойства

Значение String.

## Синтаксис

*object***.CardXML**

Свойство CardXML состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |

## Требуется подписка

Для использования этого свойства с федеральными базами требуется подписка на фичу FullAPI. Обращение к свойству без подписки приводит к неполному результату.
