# Recognize2Patterns Коллекция

## Описание

Список шаблонов для распознавания обозначений документов

## Объектная модель

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [Add](NormaCSAPI~Recognize2Patterns~Add.md) | Добавляет шаблон к списку шаблонов. Входноq параметр может быть индексом built-in-шаблона, именем создаваемого custom-шаблона, или ссылкой на существующий шаблон. |
| [AssignTo](NormaCSAPI~Recognize2Patterns~AssignTo.md) | Копирует список шаблонов в указанный |
| [Has](NormaCSAPI~Recognize2Patterns~Has.md) | Имеется ли в списке шаблонов шаблон с указанным именем или built-in индексом |
| [HasActive](NormaCSAPI~Recognize2Patterns~HasActive.md) | Имеется ли в списке шаблонов хотя бы один активный шаблон |
| [Normalize](NormaCSAPI~Recognize2Patterns~Normalize.md) | Добавляет все встроенные шаблоны распознавания к списку шаблонов и, если нет активных, делает активным Универсальный шаблон |
| [Remove](NormaCSAPI~Recognize2Patterns~Remove.md) | Удаляет шаблон распознавания из списка шаблонов |
| [RemoveAll](NormaCSAPI~Recognize2Patterns~RemoveAll.md) | Удаляет все шаблоны распознавания из списка шаблонов |
| [Test](NormaCSAPI~Recognize2Patterns~Test.md) | Разбирает проверочный текст и при успехе возвращает строку с позициями распознанных обозначения, индекса и номера |
