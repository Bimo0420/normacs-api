# SaveFolder

## Описание

Сохраняет реквизиты папки

## Возвращаемое значение

Нет.

## Синтаксис

*object***.SaveFolder** *pFolder*

Метод SaveFolder состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| pFolder | Обязательный. Объект [Folder](NormaCSAPI~Folder.md). |
