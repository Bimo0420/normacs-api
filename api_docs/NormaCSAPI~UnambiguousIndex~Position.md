# Position (UnambiguousIndex)

## Описание

Позиция шаблона в списке шаблонов, начиная с единицы

## Тип свойства

Значение Integer.

## Синтаксис

*object***.Position** [= *значение*]

Свойство Position состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndex](NormaCSAPI~UnambiguousIndex.md). |
| значение | Значение Integer. |
