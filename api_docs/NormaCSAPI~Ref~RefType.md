# RefType

## Описание

Вид связи. По умолчанию ноль ('Документ ссылается на').

## Тип свойства

Значение Integer.

## Синтаксис

*object***.RefType** [= *значение*]

Свойство RefType состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Ref](NormaCSAPI~Ref.md). |
| значение | Значение Integer. |
