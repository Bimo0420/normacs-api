# Item (Approvals)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Approval](NormaCSAPI~Approval.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Approvals](NormaCSAPI~Approvals.md). |
| Index | Обязательный. Значение Variant. |
