# Item (Amendments)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Amendment](NormaCSAPI~Amendment.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Amendments](NormaCSAPI~Amendments.md). |
| Index | Обязательный. Значение Variant. |
