# RemoveAll (UnambiguousIndexes)

## Описание

Удаляет все шаблоны распознавания из списка шаблонов

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveAll**

Метод RemoveAll состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
