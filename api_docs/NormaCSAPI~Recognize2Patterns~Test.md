# Test (Recognize2Patterns)

## Описание

Разбирает проверочный текст и при успехе возвращает строку с позициями распознанных обозначения, индекса и номера

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.Test ([TestStr])
```

Метод Test состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Patterns](NormaCSAPI~Recognize2Patterns.md). |
| TestStr | Необязательный. Значение Variant. |
