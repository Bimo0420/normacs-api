# ShowConnectionMessages

## Описание

Отображать ли всплывающие сообщения о ходе подключения к NormaCS (исключая сообщение о пересборке рабочей области, которое всегда отображается). По умолчанию False.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.ShowConnectionMessages** [= *значение*]

Свойство ShowConnectionMessages состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| значение | Значение Boolean. |
