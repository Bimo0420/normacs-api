# Execute (Find)

## Описание

Запускает поиск. Возвращает TRUE, если найден хотя бы один документ, иначе FALSE

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.Execute ([Index], [Number], [Title], [Text], [TOC], [Type], [Approver], [Approved], [Keywords], [Classifiers], [SysAdded], [SysUpdated], [Issued], [Revoked], [Changed])
```

Метод Execute состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| Index | Необязательный. Значение Variant. |
| Number | Необязательный. Значение Variant. |
| Title | Необязательный. Значение Variant. |
| Text | Необязательный. Значение Variant. |
| TOC | Необязательный. Значение Variant. |
| Type | Необязательный. Значение Variant. |
| Approver | Необязательный. Значение Variant. |
| Approved | Необязательный. Значение Variant. |
| Keywords | Необязательный. Значение Variant. |
| Classifiers | Необязательный. Значение Variant. |
| SysAdded | Необязательный. Значение Variant. |
| SysUpdated | Необязательный. Значение Variant. |
| Issued | Необязательный. Значение Variant. |
| Revoked | Необязательный. Значение Variant. |
| Changed | Необязательный. Значение Variant. |
