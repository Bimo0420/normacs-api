# ShowModal2 (RecognizeSettings)

## Описание

Показывает модальный диалог для редактирования настроек распознавания. В отличие от ShowModal, позволяет указать дескриптор родительского окна.

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.ShowModal2 (RecognizeSettings, [HwndParent])
```

Метод ShowModal2 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [RecognizeSettings](NormaCSAPI_Dialogs~RecognizeSettings.md). |
| RecognizeSettings | Обязательный. Объект Object. |
| HwndParent | Необязательный. Значение Variant. |
