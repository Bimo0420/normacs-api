# GetAmendmentByURL

## Описание

Возвращает изменение к документу с указанным адресом

## Возвращаемое значение

Объект [Amendment](NormaCSAPI~Amendment.md).

## Синтаксис

```
object.GetAmendmentByURL (URL)
```

Метод GetAmendmentByURL состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| URL | Обязательный. Значение String. |
