# Show (VerifyLinks)

## Описание

Показывает диалог

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Show** [*HwndParent*]

Метод Show состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [VerifyLinks](NormaCSAPI_Dialogs~VerifyLinks.md). |
| HwndParent | Необязательный. Значение Variant. |
