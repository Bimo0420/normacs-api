# StartLog

## Описание

Включает детализированный лог по указанному пути или в текущей папке. В лог записываются ошибки и информационные сообщения.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.StartLog** [*Path*]

Метод StartLog состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| Path | Необязательный. Значение Variant. |
