# CanProcessFile (IPackageAudit)

## Описание

Может ли быть выполнен аудит указанного файла

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.CanProcessFile (Path)
```

Метод CanProcessFile состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [IPackageAudit](NormaCSAPI~IPackageAudit.md). |
| Path | Обязательный. Значение String. |
