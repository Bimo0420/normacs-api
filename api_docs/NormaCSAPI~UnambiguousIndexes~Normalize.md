# Normalize (UnambiguousIndexes)

## Описание

Добавляет встроенные шаблоны распознавания к списку шаблонов

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Normalize**

Метод Normalize состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
