# HasDocument

## Описание

Проверяет наличие документа с указанным адресом

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.HasDocument (URL)
```

Метод HasDocument состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| URL | Обязательный. Значение String. |
