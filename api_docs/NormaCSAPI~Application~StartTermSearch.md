# StartTermSearch

## Описание

Активирует NormaCS и запускает поиск указанного термина

## Возвращаемое значение

Нет.

## Синтаксис

*object***.StartTermSearch** *TermPattern*

Метод StartTermSearch состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| TermPattern | Обязательный. Значение String. |
