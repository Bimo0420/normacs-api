# Recognize

## Описание

Возвращает объект для выполнения распознавания обозначений документов NormaCS

## Тип свойства

Объект [Recognize](NormaCSAPI~Recognize.md).

## Синтаксис

*object***.Recognize**

Свойство Recognize состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |

## Требуется ключ разработчика

Для использования этого свойства требуется ключ разработчика. Обращение к свойству без ключа вызывает ошибку или приводит к пустому результату.
