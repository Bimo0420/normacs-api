DEPRECATED Используйте объект Recognoze2 вместо этого объекта.

## Объектная модель

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [Execute](NormaCSAPI~Recognize~Execute.md) | Выполняет распознавание обозначения документа в тексте. При успешном распознавании заполняет список документов, подходящих к распознанному обозначению и записывает в NextPos позицию для следующего вызова. |
| [Reset](NormaCSAPI~Recognize~Reset.md) | Сбрасывает выходные параметры, в том числе, освобождает коллекцию документов для распознанного обозначения. Если указано, очищает кеш распознанных документов. |
