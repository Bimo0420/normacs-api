# ResetToDefaults

## Описание

Устанавливает настройки по умолчанию

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ResetToDefaults**

Метод ResetToDefaults состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
