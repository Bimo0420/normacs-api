# NextPos (Recognize)

## Описание

Позиция в тексте, с которой искать. По умолчанию перед первым запуском Execute устанавливается в единицу

## Тип свойства

Значение Integer.

## Синтаксис

*object***.NextPos** [= *значение*]

Свойство NextPos состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение Integer. |
