# IsBuiltin (Recognize2Pattern)

## Описание

Возвращает истину если шаблон является встроенным

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.IsBuiltin**

Свойство IsBuiltin состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Pattern](NormaCSAPI~Recognize2Pattern.md). |
