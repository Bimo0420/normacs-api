# AssignTo (LinkFormats)

## Описание

Копирует список форматов в указанный

## Возвращаемое значение

Нет.

## Синтаксис

*object***.AssignTo** *pVal*

Метод AssignTo состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [LinkFormats](NormaCSAPI~LinkFormats.md). |
| pVal | Обязательный. Объект [LinkFormats](NormaCSAPI~LinkFormats.md). |
