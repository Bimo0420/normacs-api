# AssignTo (Recognize2Settings)

## Описание

Копирует содержимое в указанный объект настроек

## Возвращаемое значение

Нет.

## Синтаксис

*object***.AssignTo** *pVal*

Метод AssignTo состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
| pVal | Обязательный. Объект [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
