# Seq

## Описание

Число, большее 100, определяющее относительное расположение кнопки на панели инструментов. Элемент с большим значением Seq располагается левее элемента с меньшим значением

## Тип свойства

Значение Integer.

## Синтаксис

*object***.Seq** [= *значение*]

Свойство Seq состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedAction](NormaCSAPI~UserDefinedAction.md). |
| значение | Значение Integer. |
