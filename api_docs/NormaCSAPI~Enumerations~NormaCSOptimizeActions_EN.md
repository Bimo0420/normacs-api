# NormaCSOptimizeActions

## Описание

Возможные операции при оптимизации домена.

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsOptimizeActionVacuumDatabase | 1 | Уплотнение базы данных |
|  | ncsOptimizeActionReindexDatabase | 2 | Полная переиндексация базы данных |
|  | ncsOptimizeActionDeleteEmptyFolders | 4 | Удаление пустых папок |
|  | ncsOptimizeActionReindexFTS | 8 | Индексация добавленных и измененных документов |
