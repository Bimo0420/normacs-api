# ShowModal (About)

## Описание

Показывает модальный диалог с информацией о конфигурации NormaCS и подключенных модулях

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ShowModal**

Метод ShowModal состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [About](NormaCSAPI_Dialogs~About.md). |
