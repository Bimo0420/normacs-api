# BuildDocumentURLByDomainId32

## Описание

Конструирует адрес документа по указанному домену и идентификатору в 32-ричном формате

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.BuildDocumentURLByDomainId32 (Domain, id32)
```

Метод BuildDocumentURLByDomainId32 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| Domain | Обязательный. Значение String. |
| id32 | Обязательный. Значение String. |
