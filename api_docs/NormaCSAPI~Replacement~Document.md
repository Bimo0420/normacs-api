# Document (Replacement)

## Описание

Заменяющий или заменяемый документ или Null, если документ отсутствует в базе

## Тип свойства

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

**[Set]** *object***.Document** [= *Document*]

Свойство Document состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Replacement](NormaCSAPI~Replacement.md). |
| Document | Объект [Document](NormaCSAPI~Document.md). |
