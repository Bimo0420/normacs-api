# Normalize (Recognize2Patterns)

## Описание

Добавляет все встроенные шаблоны распознавания к списку шаблонов и, если нет активных, делает активным Универсальный шаблон

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Normalize**

Метод Normalize состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Patterns](NormaCSAPI~Recognize2Patterns.md). |
