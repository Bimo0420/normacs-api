# IsBuiltin (WantRefinementIndex)

## Описание

Возвращает истину если шаблон является встроенным

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.IsBuiltin**

Свойство IsBuiltin состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [WantRefinementIndex](NormaCSAPI~WantRefinementIndex.md). |
