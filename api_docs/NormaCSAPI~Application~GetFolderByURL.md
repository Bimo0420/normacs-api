# GetFolderByURL

## Описание

Возвращает папку с указанным адресом

## Возвращаемое значение

Объект [Folder](NormaCSAPI~Folder.md).

## Синтаксис

```
object.GetFolderByURL (URL)
```

Метод GetFolderByURL состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| URL | Обязательный. Значение String. |
