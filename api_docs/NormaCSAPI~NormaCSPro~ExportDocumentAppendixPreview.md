# ExportDocumentAppendixPreview

## Описание

Выгружает файл предпросмотра приложения в указанную папку.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ExportDocumentAppendixPreview** *Document*, *Appendix*, *FolderToSave*

Метод ExportDocumentAppendixPreview состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Document | Обязательный. Объект [Document](NormaCSAPI~Document.md). |
| Appendix | Обязательный. Значение Variant. |
| FolderToSave | Обязательный. Значение String. Путь на папку для выгрузки файла. |
