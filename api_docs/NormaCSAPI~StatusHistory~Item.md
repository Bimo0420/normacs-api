# Item (StatusHistory)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [StatusHistoryItem](NormaCSAPI~StatusHistoryItem.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [StatusHistory](NormaCSAPI~StatusHistory.md). |
| Index | Обязательный. Значение Variant. |
