# HelpButtonVisible

## Описание

Видимость кнопки [Помощь]

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.HelpButtonVisible** [= *значение*]

Свойство HelpButtonVisible состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SelectDocumentForText](NormaCSAPI_Dialogs~SelectDocumentForText.md). |
| значение | Значение Boolean. |
