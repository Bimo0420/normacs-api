# Amendments

## Описание

Возвращает список изменений к документу

## Тип свойства

Объект [Amendments](NormaCSAPI~Amendments.md).

## Синтаксис

*object***.Amendments**

Свойство Amendments состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |

## Требуется подписка

Для использования этого свойства с федеральными базами требуется подписка на фичу FullAPI. Обращение к свойству без подписки вызывает ошибку или приводит к пустому результату.
