# IsMain (Revision)

## Описание

Является ли редакция основной (действующей)

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.IsMain**

Свойство IsMain состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Revision](NormaCSAPI~Revision.md). |

## Замечания

Этот атрибут нельзя редактировать, он обновляется автоматически при изменении типа редакции.
