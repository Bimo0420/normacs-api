# Clone (WantRefinementIndex)

## Описание

Создает копию текущего шаблона (не включенную в список шаблонов)

## Возвращаемое значение

Объект [WantRefinementIndex](NormaCSAPI~WantRefinementIndex.md).

## Синтаксис

*object***.Clone**

Метод Clone состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [WantRefinementIndex](NormaCSAPI~WantRefinementIndex.md). |
