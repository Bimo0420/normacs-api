# Add (LinkFormats)

## Описание

Добавляет формат

## Возвращаемое значение

Объект [LinkFormat](NormaCSAPI~LinkFormat.md).

## Синтаксис

```
object.Add (Name, Format)
```

Метод Add состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [LinkFormats](NormaCSAPI~LinkFormats.md). |
| Name | Обязательный. Значение String. |
| Format | Обязательный. Значение String. |
