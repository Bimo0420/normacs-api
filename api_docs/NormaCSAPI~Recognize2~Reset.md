# Reset (Recognize2)

## Описание

Сбрасывает выходные параметры, в том числе, освобождает коллекцию документов для распознанного обозначения. Если указано, очищает кеш распознанных документов.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Reset** [*varClearDocumentCache*]

Метод Reset состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2](NormaCSAPI~Recognize2.md). |
| varClearDocumentCache | Необязательный. Значение Variant. |
