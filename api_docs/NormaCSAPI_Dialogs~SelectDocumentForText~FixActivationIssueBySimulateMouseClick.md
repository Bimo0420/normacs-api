# FixActivationIssueBySimulateMouseClick

## Описание

Исправить ошибку начальной активации диалога путем симуляции щелчка мышкой на выбранном элементе списка

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.FixActivationIssueBySimulateMouseClick** [= *значение*]

Свойство FixActivationIssueBySimulateMouseClick состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SelectDocumentForText](NormaCSAPI_Dialogs~SelectDocumentForText.md). |
| значение | Значение Boolean. |
