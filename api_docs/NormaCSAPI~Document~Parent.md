# Parent (Document)

## Описание

Возвращает ссылку на сборник, которому принадлежит документ, или нулевое значение, если документ не входит ни в какой в сборник

## Тип свойства

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

*object***.Parent**

Свойство Parent состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
