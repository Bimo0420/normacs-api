# RenameOrganization

## Описание

Переименовывает указанную организацию.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RenameOrganization** *Organization*, *NewName*

Метод RenameOrganization состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Organization | Обязательный. Значение Variant. |
| NewName | Обязательный. Значение String. |
