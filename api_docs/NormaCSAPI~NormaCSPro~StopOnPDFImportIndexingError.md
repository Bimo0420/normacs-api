# StopOnPDFImportIndexingError

## Описание

Прерывать загрузку PDF-документа и возбуждать исключение при ошибке индексирования

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.StopOnPDFImportIndexingError** [= *значение*]

Свойство StopOnPDFImportIndexingError состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| значение | Значение Boolean. |
