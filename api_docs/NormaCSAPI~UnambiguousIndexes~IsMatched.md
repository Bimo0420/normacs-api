# IsMatched

## Описание

Является ли указанный индекс неоднозначным

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.IsMatched (Index)
```

Метод IsMatched состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
| Index | Обязательный. Значение String. |
