# Add (UserDefinedActions)

## Описание

Добавляет указанный элемент (документ) в коллекцию

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Add** *pVal*

Метод Add состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
| pVal | Обязательный. Объект [UserDefinedAction](NormaCSAPI~UserDefinedAction.md). |
