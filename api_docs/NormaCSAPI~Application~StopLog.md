# StopLog

## Описание

Выключает детализированный лог, в лог записываются только ошибки.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.StopLog**

Метод StopLog состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
