# CopyHyperlinkToClipboard

## Описание

Копирует ссылку на документ в буфер обмена

## Возвращаемое значение

Нет.

## Синтаксис

*object***.CopyHyperlinkToClipboard**

Метод CopyHyperlinkToClipboard состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
