# Remove (UnambiguousIndexes)

## Описание

Удаляет шаблон распознавания из списка шаблонов

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Remove** *Index*

Метод Remove состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
| Index | Обязательный. Значение Variant. |
