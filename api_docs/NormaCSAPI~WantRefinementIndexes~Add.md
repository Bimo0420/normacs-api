# Add (WantRefinementIndexes)

## Описание

Добавляет шаблон к списку шаблонов. Входноq параметр может быть индексом built-in-шаблона, именем создаваемого custom-шаблона, или ссылкой на существующий шаблон.

## Возвращаемое значение

Объект [WantRefinementIndex](NormaCSAPI~WantRefinementIndex.md).

## Синтаксис

```
object.Add (BuiltinIndexOrNameOrRef)
```

Метод Add состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [WantRefinementIndexes](NormaCSAPI~WantRefinementIndexes.md). |
| BuiltinIndexOrNameOrRef | Обязательный. Значение Variant. |
