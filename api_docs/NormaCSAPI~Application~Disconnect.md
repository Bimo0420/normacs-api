# Disconnect

## Описание

Закрывает соединение с подключенными базами данных. При работе в серверной версии NormaCS с ограниченным числом пользователей, рекомендуется вызывать этот метод каждый раз, когда работа с NormaCS API временно завершена, но объект API не планируется освобождать

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Disconnect**

Метод Disconnect состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
