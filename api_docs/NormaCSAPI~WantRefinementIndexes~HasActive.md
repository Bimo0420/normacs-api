# HasActive (WantRefinementIndexes)

## Описание

Имеется ли в списке шаблонов хотя бы один активный шаблон

## Возвращаемое значение

Значение Boolean.

## Синтаксис

*object***.HasActive**

Метод HasActive состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [WantRefinementIndexes](NormaCSAPI~WantRefinementIndexes.md). |
