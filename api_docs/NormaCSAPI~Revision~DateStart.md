# DateStart (Revision)

## Описание

Дата начала действия редакции. Для изменения используйте метод NormaCSPro.RevisionEditor.UpdateRevision.

## Тип свойства

Значение Date.

## Синтаксис

*object***.DateStart**

Свойство DateStart состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Revision](NormaCSAPI~Revision.md). |

## Замечания

В списке редакций приложения NormaCS Pro этому атрибуту соответствует поле "Дата начала".
