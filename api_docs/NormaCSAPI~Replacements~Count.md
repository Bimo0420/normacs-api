# Count (Replacements)

## Описание

Возвращает количество элементов в коллекции

## Тип свойства

Значение Long.

## Синтаксис

*object***.Count**

Свойство Count состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Replacements](NormaCSAPI~Replacements.md). |
