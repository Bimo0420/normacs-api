# AddKeyword

## Описание

Добавляет указанное ключевое слово к справочнику ключевых слов домена

## Возвращаемое значение

Объект [Keyword](NormaCSAPI~Keyword.md).

## Синтаксис

```
object.AddKeyword (Name)
```

Метод AddKeyword состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Name | Обязательный. Значение String. |
