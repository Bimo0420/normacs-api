# StartDocumentSearchByText

## Описание

Активирует NormaCS и запускает поиск документов с указанным текстом

## Возвращаемое значение

Нет.

## Синтаксис

*object***.StartDocumentSearchByText** *DocumentTextPattern*

Метод StartDocumentSearchByText состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| DocumentTextPattern | Обязательный. Значение String. |
