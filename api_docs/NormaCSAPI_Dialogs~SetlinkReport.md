| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [Create](NormaCSAPI_Dialogs~SetlinkReport~Create.md) | Создает или пересоздает диалог для указанной сессии расстановки ссылок или анализа документа |
| [Destroy](NormaCSAPI_Dialogs~SetlinkReport~Destroy.md) | Уничтожает диалог |
| [Hide](NormaCSAPI_Dialogs~SetlinkReport~Hide.md) | Скрывает диалог |
| [Show](NormaCSAPI_Dialogs~SetlinkReport~Show.md) | Показывает диалог |
