# GetURLType

## Описание

Возвращает тип объекта NormaCS по указанному адресу

## Возвращаемое значение

Перечисление [NormaCSURLType](NormaCSAPI~Enumerations~NormaCSURLType_EN.md).

## Синтаксис

```
object.GetURLType (URL)
```

Метод GetURLType состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| URL | Обязательный. Значение String. |
