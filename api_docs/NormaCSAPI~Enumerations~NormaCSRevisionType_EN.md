# NormaCSRevisionType

## Описание

Тип редакции: основная, прошлая, первоначальная или будущая.

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsRevisionTypeMain | 0 | Основная (действующая) редакция документа |
|  | ncsRevisionTypePast | 1 | Прошлая (архивная) редакция документа |
|  | ncsRevisionTypeInitial | 2 | Первоначальная редакция документа |
|  | ncsRevisionTypeFuture | 3 | Будущая (предстоящая) редакция документа |
