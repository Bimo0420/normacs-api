# NormaCSDocumentImageFormat

## Описание

Формат изображения документа

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsDocumentImageFormatUnknown | 0 | Формат недоступен или нераспознан |
|  | ncsDocumentImageFormatTiff | 1 | Формат TIFF |
|  | ncsDocumentImageFormatPdf | 2 | Формат PDF |
|  | ncsDocumentImageFormatDjvu | 3 | Формат DjVu |
|  | ncsDocumentImageFormatDwf | 4 | Формат DWF (Autodesk) |
|  | ncsDocumentImageFormatPdfByNormaCS | 5 | Формат PDF, при этом документ отображается средствами NormaCS, что предоставляет расширенные возможности контроля доступа |
|  | ncsDocumentImageFormatDjvuByNormaCS | 6 | Формат DjVu, при этом документ отображается средствами NormaCS, что предоставляет расширенные возможности контроля доступа |
