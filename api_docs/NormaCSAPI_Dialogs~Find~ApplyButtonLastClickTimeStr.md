# ApplyButtonLastClickTimeStr

## Описание

Строка, содержащая время последнего вызова пункта меню [Вставить] в формате YYYY-MM-DD HH:MM:SS. Может быть использовано для отслеживания вызовов без помощи обработчика событий.

## Тип свойства

Значение String.

## Синтаксис

*object***.ApplyButtonLastClickTimeStr**

Свойство ApplyButtonLastClickTimeStr состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
