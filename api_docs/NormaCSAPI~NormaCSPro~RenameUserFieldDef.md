# RenameUserFieldDef

## Описание

Переименовывает указанное определение пользовательского поля

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RenameUserFieldDef** *p*, *NewName*

Метод RenameUserFieldDef состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| p | Обязательный. Объект [UserFieldDef](NormaCSAPI~UserFieldDef.md). |
| NewName | Обязательный. Значение String. |
