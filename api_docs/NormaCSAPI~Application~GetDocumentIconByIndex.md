# GetDocumentIconByIndex

## Описание

Возвращает дескриптор значка документа (HICON) по его внутреннему индексу,

## Возвращаемое значение

Значение Long.

## Синтаксис

```
object.GetDocumentIconByIndex (IconIndex)
```

Метод GetDocumentIconByIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| IconIndex | Обязательный. Значение Integer. |
