# IsStatusChangedBetweenDates

## Описание

Изменился ли статус документа в указанном итервале дат

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.IsStatusChangedBetweenDates (date1, date2)
```

Метод IsStatusChangedBetweenDates состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
| date1 | Обязательный. Значение Date. |
| date2 | Обязательный. Значение Date. |
