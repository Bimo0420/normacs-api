# GetOemInfoString

## Описание

Возвращает указанное именованное значение из конфигурационного файла oem.inf, или пустую строку, если не найдено

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetOemInfoString (Section, Name)
```

Метод GetOemInfoString состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| Section | Обязательный. Значение String. |
| Name | Обязательный. Значение String. |
