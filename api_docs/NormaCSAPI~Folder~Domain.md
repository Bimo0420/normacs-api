# Domain (Folder)

## Описание

Возвращает имя домена, к которому принадлежит данная папка

## Тип свойства

Значение String.

## Синтаксис

*object***.Domain**

Свойство Domain состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Folder](NormaCSAPI~Folder.md). |
