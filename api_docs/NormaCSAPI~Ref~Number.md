# Number (Ref)

## Описание

Номер документа. Указывается, если документ отсутствует в NormaCS.

## Тип свойства

Значение String.

## Синтаксис

*object***.Number** [= *значение*]

Свойство Number состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Ref](NormaCSAPI~Ref.md). |
| значение | Значение String. |
