# GetNormaCSPro

## Описание

Возвращает вспомогательный объект для редактирования данных домена.

## Возвращаемое значение

Объект [NormaCSPro](NormaCSAPI~NormaCSPro.md).

## Синтаксис

```
object.GetNormaCSPro (password)
```

Метод GetNormaCSPro состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domain](NormaCSAPI~Domain.md). |
| password | Обязательный. Значение String. |

## Требуется подписка

Для использования этого метода требуется подписка на фичу APIPro.
