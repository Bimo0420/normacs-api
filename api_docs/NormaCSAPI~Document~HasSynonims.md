# HasSynonims

## Описание

Имеет ли документ дополнительные обозначения. В этом случае список Document.Synonims содержит одну или несколько записей с IsMain=False.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.HasSynonims**

Свойство HasSynonims состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
