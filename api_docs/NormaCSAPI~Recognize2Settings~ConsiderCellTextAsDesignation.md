# ConsiderCellTextAsDesignation

## Описание

Пытаться распознать текст внутри ячейки таблицы как одно целое обозначение

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.ConsiderCellTextAsDesignation** [= *значение*]

Свойство ConsiderCellTextAsDesignation состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
| значение | Значение Boolean. |
