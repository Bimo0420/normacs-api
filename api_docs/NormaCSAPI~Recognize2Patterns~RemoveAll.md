# RemoveAll (Recognize2Patterns)

## Описание

Удаляет все шаблоны распознавания из списка шаблонов

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveAll**

Метод RemoveAll состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Patterns](NormaCSAPI~Recognize2Patterns.md). |
