# ShowModal (RecognizeSettings)

## Описание

Показывает модальный диалог для редактирования настроек распознавания

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.ShowModal (RecognizeSettings)
```

Метод ShowModal состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [RecognizeSettings](NormaCSAPI_Dialogs~RecognizeSettings.md). |
| RecognizeSettings | Обязательный. Объект Object. |
