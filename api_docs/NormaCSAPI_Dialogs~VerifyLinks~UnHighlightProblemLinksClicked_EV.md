# UnHighlightProblemLinksClicked Событие

## Описание

Нажата кнопка "убрать выделение проблемных ссылок"

## Синтаксис

```
Private Sub object_UnHighlightProblemLinksClicked(Filtered)
```

Событие UnHighlightProblemLinksClicked состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект [VerifyLinks](NormaCSAPI_Dialogs~VerifyLinks.md). |
| Filtered | Значение Boolean. |
