# Keywords (Find)

## Описание

Список ключевых слов разыскиваемых документов, разделенных символом точки с запятой (;)

## Тип свойства

Значение String.

## Синтаксис

*object***.Keywords** [= *значение*]

Свойство Keywords состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| значение | Значение String. |
