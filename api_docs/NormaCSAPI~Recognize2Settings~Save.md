# Save

## Описание

Сохраняет настройки в файле по указанному пути

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Save** *bstrPath*

Метод Save состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
| bstrPath | Обязательный. Значение String. |
