# WantRefinementIndexes Коллекция

## Описание

Список шаблонов достоверных индексов

## Объектная модель

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [Add](NormaCSAPI~WantRefinementIndexes~Add.md) | Добавляет шаблон к списку шаблонов. Входноq параметр может быть индексом built-in-шаблона, именем создаваемого custom-шаблона, или ссылкой на существующий шаблон. |
| [AssignTo](NormaCSAPI~WantRefinementIndexes~AssignTo.md) | Копирует список шаблонов в указанный |
| [Has](NormaCSAPI~WantRefinementIndexes~Has.md) | Имеется ли в списке шаблонов шаблон с указанным именем или built-in индексом |
| [HasActive](NormaCSAPI~WantRefinementIndexes~HasActive.md) | Имеется ли в списке шаблонов хотя бы один активный шаблон |
| [Normalize](NormaCSAPI~WantRefinementIndexes~Normalize.md) | Добавляет встроенные шаблоны распознавания к списку шаблонов |
| [Remove](NormaCSAPI~WantRefinementIndexes~Remove.md) | Удаляет шаблон распознавания из списка шаблонов |
| [RemoveAll](NormaCSAPI~WantRefinementIndexes~RemoveAll.md) | Удаляет все шаблоны распознавания из списка шаблонов |
| [Test](NormaCSAPI~WantRefinementIndexes~Test.md) |  |
