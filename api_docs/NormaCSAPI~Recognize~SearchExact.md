# SearchExact

## Описание

Искать только документы, имеющие обозначения, совпадающие с распознанным. По умолчанию не установлено.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.SearchExact** [= *значение*]

Свойство SearchExact состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение Boolean. |
