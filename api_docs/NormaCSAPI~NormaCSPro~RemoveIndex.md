# RemoveIndex

## Описание

Удаляет указанный индекс из списка индексов домена

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveIndex** *Index*

Метод RemoveIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Index | Обязательный. Значение Variant. |
