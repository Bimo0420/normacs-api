# Item (Synonims)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Synonim](NormaCSAPI~Synonim.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Synonims](NormaCSAPI~Synonims.md). |
| Index | Обязательный. Значение Variant. |
