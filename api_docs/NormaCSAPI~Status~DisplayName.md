# DisplayName

## Описание

Наименование статуса, предваренное наименованием группы; для группы [Не определен] только наименование статуса

## Тип свойства

Значение String.

## Синтаксис

*object***.DisplayName**

Свойство DisplayName состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Status](NormaCSAPI~Status.md). |
