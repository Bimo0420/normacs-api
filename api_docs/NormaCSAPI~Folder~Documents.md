# Documents (Folder)

## Описание

Возвращает список документов из папки (не включая вложенные папки)

## Тип свойства

Объект [Documents](NormaCSAPI~Documents.md).

## Синтаксис

*object***.Documents**

Свойство Documents состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Folder](NormaCSAPI~Folder.md). |
