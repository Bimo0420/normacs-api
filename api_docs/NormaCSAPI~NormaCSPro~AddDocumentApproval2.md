# AddDocumentApproval2

## Описание

Добавляет утверждение к списку утверждений документа. Данные об утверждении указываются через ссылку на утверждающий документ. Вызовите SaveDocument для сохранения в базе измененных атрибутов документа.

## Возвращаемое значение

Объект [Approval](NormaCSAPI~Approval.md).

## Синтаксис

```
object.AddDocumentApproval2 (pDocument, ApprovingDocument, IsMain)
```

Метод AddDocumentApproval2 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| pDocument | Обязательный. Объект [Document](NormaCSAPI~Document.md). Документ, к которому добавляется утверждение. |
| ApprovingDocument | Обязательный. Значение Variant. Ссылка на утверждающий документ. Обратите внимание, что утверждающий и утверждаемый документы должны принадлежать к одному и тому же домену. |
| IsMain | Обязательный. Значение Boolean. Сделать ли добавляемое утверждение основным в списке утверждений. |

## Замечания

Обратите внимание, что утверждающий и утверждаемый документы должны принадлежать к одному и тому же домену.
