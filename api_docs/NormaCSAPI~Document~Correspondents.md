# Correspondents

## Описание

Возвращает список документов, содержащих ссылки на этот документ (корреспонденты)

## Тип свойства

Объект [Documents](NormaCSAPI~Documents.md).

## Синтаксис

*object***.Correspondents**

Свойство Correspondents состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
