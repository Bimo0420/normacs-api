# Index (Find)

## Описание

Образец индекса разыскиваемых документов. Может включать групповой символ (*).

## Тип свойства

Значение String.

## Синтаксис

*object***.Index** [= *значение*]

Свойство Index состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| значение | Значение String. |
