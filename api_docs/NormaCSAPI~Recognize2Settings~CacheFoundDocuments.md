# CacheFoundDocuments

## Описание

Кешировать коллекции документов для распознанных обозначений. Ускоряет поиск, но может потребовать большое количество оперативной памяти. По умолчанию не установлено.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.CacheFoundDocuments** [= *значение*]

Свойство CacheFoundDocuments состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
| значение | Значение Boolean. |
