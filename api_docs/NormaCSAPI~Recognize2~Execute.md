# Execute (Recognize2)

## Описание

Выполняет распознавание обозначения документа в тексте. При успешном распознавании обновляет NextPos, и, если указано, формирует список документов, подходящих к распознанному обозначению.

## Возвращаемое значение

Значение Boolean.

## Синтаксис

*object***.Execute**

Метод Execute состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2](NormaCSAPI~Recognize2.md). |

## Замечания

Перед первым вызовом следует настроить шаблоны распознавания, например, при помощи Recognize2.Settings.Patterns.Normalize.
