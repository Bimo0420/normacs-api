# ExportDocumentAppendix2

## Описание

Выгружает приложение к документу. В отличие от ExportDocumentAppendix, возвращает путь на выгруженный файл.

## Возвращаемое значение

Значение String. Путь на выгруженный файл.

## Синтаксис

```
object.ExportDocumentAppendix2 (Document, Appendix, FolderToSave)
```

Метод ExportDocumentAppendix2 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Document | Обязательный. Объект [Document](NormaCSAPI~Document.md). |
| Appendix | Обязательный. Значение Variant. |
| FolderToSave | Обязательный. Значение String. |
