# IsDocumentToolbar

## Описание

Поместить на панель инструментов документа, а не на основную панель инструментов

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.IsDocumentToolbar** [= *значение*]

Свойство IsDocumentToolbar состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedAction](NormaCSAPI~UserDefinedAction.md). |
| значение | Значение Boolean. |
