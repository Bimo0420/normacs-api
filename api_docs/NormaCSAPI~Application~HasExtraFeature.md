# HasExtraFeature

## Описание

Имеется ли в приобретенном комплекте feature с указанным именем

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.HasExtraFeature (FeatureName)
```

Метод HasExtraFeature состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| FeatureName | Обязательный. Значение String. |
