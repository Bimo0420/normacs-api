# RemoveFolder

## Описание

Удаляет указанную папку

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveFolder** *Folder*

Метод RemoveFolder состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Folder | Обязательный. Значение Variant. |
