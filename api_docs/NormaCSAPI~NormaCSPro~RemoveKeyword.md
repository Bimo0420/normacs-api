# RemoveKeyword

## Описание

Удаляет указанное ключевое слово из справочника ключевых слов домена

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveKeyword** *Keyword*

Метод RemoveKeyword состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Keyword | Обязательный. Значение Variant. |
