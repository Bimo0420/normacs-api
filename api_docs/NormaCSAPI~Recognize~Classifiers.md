# Classifiers (Recognize)

## Описание

Список классификаторов разыскиваемых документов, разделенных символом точки с запятой

## Тип свойства

Значение String.

## Синтаксис

*object***.Classifiers** [= *значение*]

Свойство Classifiers состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение String. |
