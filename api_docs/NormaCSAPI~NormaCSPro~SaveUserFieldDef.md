# SaveUserFieldDef

## Описание

Сохраняет определение пользовательского поля

## Возвращаемое значение

Нет.

## Синтаксис

*object***.SaveUserFieldDef** *p*

Метод SaveUserFieldDef состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| p | Обязательный. Объект [UserFieldDef](NormaCSAPI~UserFieldDef.md). |
