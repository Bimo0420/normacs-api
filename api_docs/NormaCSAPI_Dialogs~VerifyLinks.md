# VerifyLinks Объект

## Описание

Диалог проверки ссылок

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [AddLink](NormaCSAPI_Dialogs~VerifyLinks~AddLink.md) | Добавляет ссылку для проверки |
| [ClearLinks](NormaCSAPI_Dialogs~VerifyLinks~ClearLinks.md) | Удаляет ссылки |
| [Create](NormaCSAPI_Dialogs~VerifyLinks~Create.md) | Создает или пересоздает диалог проверки ссылок |
| [Destroy](NormaCSAPI_Dialogs~VerifyLinks~Destroy.md) | Уничтожает диалог |
| [Hide](NormaCSAPI_Dialogs~VerifyLinks~Hide.md) | Скрывает диалог |
| [Refresh](NormaCSAPI_Dialogs~VerifyLinks~Refresh.md) | Обновляет окно диалога в соответствии с указанным путем на документ и списком ссылок |
| [Show](NormaCSAPI_Dialogs~VerifyLinks~Show.md) | Показывает диалог |
