Возвращает список разработчиков документа

## Тип свойства

Объект [Developers](NormaCSAPI~Developers.md).

## Синтаксис

*object***.Developers**

Свойство Developers состоит из:

| Часть | Описание |
| --- | --- |

## Требуется подписка

Для использования этого свойства с федеральными базами требуется подписка на фичу FullAPI. Обращение к свойству без подписки вызывает ошибку или приводит к пустому результату.
