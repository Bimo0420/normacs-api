# Count (Documents)

## Описание

Возвращает количество элементов (документов) в коллекции

## Тип свойства

Значение Long.

## Синтаксис

*object***.Count**

Свойство Count состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Documents](NormaCSAPI~Documents.md). |
