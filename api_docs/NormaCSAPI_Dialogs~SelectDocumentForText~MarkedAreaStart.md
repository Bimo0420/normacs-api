# MarkedAreaStart

## Описание

Начальная позиция подкрашенной части текста начиная с единицы или ноль для всего текста

## Тип свойства

Значение Integer.

## Синтаксис

*object***.MarkedAreaStart** [= *значение*]

Свойство MarkedAreaStart состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SelectDocumentForText](NormaCSAPI_Dialogs~SelectDocumentForText.md). |
| значение | Значение Integer. |
