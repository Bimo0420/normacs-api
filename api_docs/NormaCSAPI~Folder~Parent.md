# Parent (Folder)

## Описание

Ссылка на объект родительской папки или нулевой указатель если папка верхнего уровня

## Тип свойства

Объект [Folder](NormaCSAPI~Folder.md).

## Синтаксис

**[Set]** *object***.Parent** [= *Folder*]

Свойство Parent состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Folder](NormaCSAPI~Folder.md). |
| Folder | Объект [Folder](NormaCSAPI~Folder.md). |
