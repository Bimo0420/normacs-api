## Методы и свойства

| Свойства |  | Свойства |
| --- | --- | --- |
|  | Свойства |
| [Classifier](NormaCSAPI~Folder~Classifier.md) |  | Классификатор папки |
| [Comment](NormaCSAPI~Folder~Comment.md) |  | Комментарий к папке |
| [Documents](NormaCSAPI~Folder~Documents.md) |  | Возвращает список документов из папки (не включая вложенные папки) |
| [Domain](NormaCSAPI~Folder~Domain.md) |  | Возвращает имя домена, к которому принадлежит данная папка |
| [Folders](NormaCSAPI~Folder~Folders.md) |  | Возвращает список дочерних папок |
| [Id](NormaCSAPI~Folder~Id.md) |  | Возвращает идентификатор папки внутри домена |
| [Name](NormaCSAPI~Folder~Name.md) |  | Название папки |
| [Parent](NormaCSAPI~Folder~Parent.md) |  | Ссылка на объект родительской папки или нулевой указатель если папка верхнего уровня |
| [Path](NormaCSAPI~Folder~Path.md) |  | Возвращает полный путь на папку |
| [Sort](NormaCSAPI~Folder~Sort.md) |  | Число, определяющее относительное положение папки в списке папок родителя |
| [URL](NormaCSAPI~Folder~URL.md) |  | Возвращает URL папки |
