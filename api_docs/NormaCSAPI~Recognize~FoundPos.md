# FoundPos (Recognize)

## Описание

Позиция распознанного обозначения в исходном тексте, начиная с единицы, или ноль, если не найлено

## Тип свойства

Значение Integer.

## Синтаксис

*object***.FoundPos**

Свойство FoundPos состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
