## Тип свойства

Значение Integer.

## Синтаксис

*object***.PageCount** [= *значение*]

Свойство PageCount состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
| значение | Значение Integer. |
