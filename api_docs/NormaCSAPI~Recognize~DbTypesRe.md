# DbTypesRe

## Описание

Вспомогательный метод, возвращающий регулярное выражение для распознавания индексов подключенных доменов

## Тип свойства

Значение String.

## Синтаксис

*object***.DbTypesRe**

Свойство DbTypesRe состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
