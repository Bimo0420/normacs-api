# Item (Indexes)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Index](NormaCSAPI~Index.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Indexes](NormaCSAPI~Indexes.md). |
| Index | Обязательный. Значение Variant. |
