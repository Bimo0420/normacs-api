# GetLogAsXml

## Описание

Возвращает xml-представление протокола указанной сессии расстановки ссылок

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetLogAsXml (SessionId)
```

Метод GetLogAsXml состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| SessionId | Обязательный. Значение String. |
