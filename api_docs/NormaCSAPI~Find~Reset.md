# Reset (Find)

## Описание

Сбрасывает значения параметров, относящихся к образцам поиска и очищает список найденных документов. Параметры настройки поиска, такие как FindByTextMode и Limit, при этом не изменяются.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Reset**

Метод Reset состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
