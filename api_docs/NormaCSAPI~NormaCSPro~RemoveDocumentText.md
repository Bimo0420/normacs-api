# RemoveDocumentText

## Описание

Удаляет текст документа

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveDocumentText** *Document*

Метод RemoveDocumentText состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Document | Обязательный. Значение Variant. |
