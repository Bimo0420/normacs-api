# Item (Refs)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Ref](NormaCSAPI~Ref.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Refs](NormaCSAPI~Refs.md). |
| Index | Обязательный. Значение Variant. |
