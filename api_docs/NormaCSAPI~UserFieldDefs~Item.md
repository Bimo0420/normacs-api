# Item (UserFieldDefs)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [UserFieldDef](NormaCSAPI~UserFieldDef.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserFieldDefs](NormaCSAPI~UserFieldDefs.md). |
| Index | Обязательный. Значение Variant. |
