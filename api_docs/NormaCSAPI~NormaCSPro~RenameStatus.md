# RenameStatus

## Описание

Переименовывает указанный статус

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RenameStatus** *Status*, *NewName*

Метод RenameStatus состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Status | Обязательный. Значение Variant. |
| NewName | Обязательный. Значение String. |
