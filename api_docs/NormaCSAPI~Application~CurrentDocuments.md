# CurrentDocuments

## Описание

Возвращает текущую выборку документов NormaCS. Предварительно следует запустить NormaCS, отметить галочками нужные документы и выбрать в контекстном меню списка [Копировать ссылку...]. Если ни один документ не выбран, то возвращается вся выборка.

## Тип свойства

Объект [Documents](NormaCSAPI~Documents.md).

## Синтаксис

*object***.CurrentDocuments**

Свойство CurrentDocuments состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
