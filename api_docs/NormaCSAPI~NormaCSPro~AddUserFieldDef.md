# AddUserFieldDef

## Описание

Создает определение пользовательского поля

## Возвращаемое значение

Объект [UserFieldDef](NormaCSAPI~UserFieldDef.md).

## Синтаксис

```
object.AddUserFieldDef (Name, [Type], [Length], [GroupName], [Comment])
```

Метод AddUserFieldDef состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Name | Обязательный. Значение String. Название приложения. |
| Type | Необязательный. Значение Variant. |
| Length | Необязательный. Значение Variant. |
| GroupName | Необязательный. Значение Variant. |
| Comment | Необязательный. Значение Variant. |
