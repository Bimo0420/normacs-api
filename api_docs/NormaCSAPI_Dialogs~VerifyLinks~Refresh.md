# Refresh

## Описание

Обновляет окно диалога в соответствии с указанным путем на документ и списком ссылок

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Refresh**

Метод Refresh состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [VerifyLinks](NormaCSAPI_Dialogs~VerifyLinks.md). |
