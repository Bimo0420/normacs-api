# DestroyDesignationCache

## Описание

Убирает кеш для поддержки поиска документов по обозначениям

## Возвращаемое значение

Нет.

## Синтаксис

*object***.DestroyDesignationCache**

Метод DestroyDesignationCache состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
