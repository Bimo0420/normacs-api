# RemoveUserFieldDef

## Описание

Удаляет указанное определение пользовательского поля

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveUserFieldDef** *p*

Метод RemoveUserFieldDef состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| p | Обязательный. Объект [UserFieldDef](NormaCSAPI~UserFieldDef.md). |
