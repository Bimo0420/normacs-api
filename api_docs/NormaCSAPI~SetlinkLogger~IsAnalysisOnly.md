# IsAnalysisOnly

## Описание

Указанная сессия предназначена только для анализа документа

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.IsAnalysisOnly (SessionId)
```

Метод IsAnalysisOnly состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| SessionId | Обязательный. Значение String. |
