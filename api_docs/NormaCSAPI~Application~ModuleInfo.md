# ModuleInfo

## Описание

Возвращает объект для получения информации о версии библиотеки и местоположении и зависимых модулей

## Тип свойства

Объект [ModuleInfo](NormaCSAPI~ModuleInfo.md).

## Синтаксис

*object***.ModuleInfo**

Свойство ModuleInfo состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
