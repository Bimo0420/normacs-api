# Count (UnambiguousIndexes)

## Описание

Возвращает количество элементов в списке шаблонов

## Тип свойства

Значение Long.

## Синтаксис

*object***.Count**

Свойство Count состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
