# Item (Agreements)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Agreement](NormaCSAPI~Agreement.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Agreements](NormaCSAPI~Agreements.md). |
| Index | Обязательный. Значение Variant. |
