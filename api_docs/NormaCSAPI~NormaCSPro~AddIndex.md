# AddIndex

## Описание

Добавляет указанный индекс к списку индексов домена

## Возвращаемое значение

Объект [Index](NormaCSAPI~Index.md).

## Синтаксис

```
object.AddIndex (Name)
```

Метод AddIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Name | Обязательный. Значение String. |
