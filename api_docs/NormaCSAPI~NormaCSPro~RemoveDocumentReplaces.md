# RemoveDocumentReplaces

## Описание

Удаляет указанную замену. Вызовите SaveDocument для сохранения в базе измененных атрибутов документа.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveDocumentReplaces** *pDocument*, *Replacement*

Метод RemoveDocumentReplaces состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| pDocument | Обязательный. Объект [Document](NormaCSAPI~Document.md). |
| Replacement | Обязательный. Значение Variant. |
