# NormaCSFindByTextMode

## Описание

Модификатор поиска по тексту документа

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsFindByTextAnyWord | 0 | Искать любую комбинацию из указанных слов (установлено по умолчанию) |
|  | ncsFindByTextAllWordsInAnyOrder | 1 | Искать все указанные слова в любом порядке |
|  | ncsFindByTextAllWordsInSpecifiedOrder | 2 | Искать все указанные слова в указанном порядке |
