# HasActive (UnambiguousIndexes)

## Описание

Имеется ли в списке шаблонов хотя бы один активный шаблон

## Возвращаемое значение

Значение Boolean.

## Синтаксис

*object***.HasActive**

Метод HasActive состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
