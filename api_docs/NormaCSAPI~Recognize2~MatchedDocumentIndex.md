# MatchedDocumentIndex (Recognize2)

## Описание

Индекс документа в списке Documents, обозначение которого совпадает с распознанным (с точностью до разделителей). Начинается с единицы.

## Тип свойства

Значение Integer.

## Синтаксис

*object***.MatchedDocumentIndex**

Свойство MatchedDocumentIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2](NormaCSAPI~Recognize2.md). |
