# Item (Folders)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Folder](NormaCSAPI~Folder.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Folders](NormaCSAPI~Folders.md). |
| Index | Обязательный. Значение Variant. |
