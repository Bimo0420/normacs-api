# Create (SetlinkReport)

## Описание

Создает или пересоздает диалог для указанной сессии расстановки ссылок или анализа документа

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Create** *SessionId*

Метод Create состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkReport](NormaCSAPI_Dialogs~SetlinkReport.md). |
| SessionId | Обязательный. Значение String. |
