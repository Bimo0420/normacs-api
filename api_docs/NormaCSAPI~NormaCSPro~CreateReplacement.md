# CreateReplacement

## Описание

Создает объект Replacement для использования в функциях добавления к документу данных о замене

## Возвращаемое значение

Объект [Replacement](NormaCSAPI~Replacement.md).

## Синтаксис

```
object.CreateReplacement ([Document], [ReplacementType])
```

Метод CreateReplacement состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Document | Необязательный. Значение Variant. |
| ReplacementType | Необязательный. Значение Variant. |
