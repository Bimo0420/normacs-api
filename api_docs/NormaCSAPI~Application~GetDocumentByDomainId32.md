# GetDocumentByDomainId32

## Описание

Возвращает документ с указанным доменом и идентификатором в 32-ричном формате

## Возвращаемое значение

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

```
object.GetDocumentByDomainId32 (Domain, id32)
```

Метод GetDocumentByDomainId32 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| Domain | Обязательный. Значение String. |
| id32 | Обязательный. Значение String. |
