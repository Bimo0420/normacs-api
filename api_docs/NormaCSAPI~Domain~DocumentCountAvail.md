# DocumentCountAvail

## Описание

Возвращает количество доступных для чтения документов для домена, то есть документы, имеющие текст и/или изображение, и принадлежащие приобретенным разделам

## Тип свойства

Значение Long.

## Синтаксис

*object***.DocumentCountAvail**

Свойство DocumentCountAvail состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domain](NormaCSAPI~Domain.md). |
