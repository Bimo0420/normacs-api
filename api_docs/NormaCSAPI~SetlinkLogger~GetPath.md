# GetPath

## Описание

Возвращает путь на документ для указанной сессии расстановки ссылок

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetPath (SessionId)
```

Метод GetPath состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SetlinkLogger](NormaCSAPI~SetlinkLogger.md). |
| SessionId | Обязательный. Значение String. |
