Подстрока оглавления разыскиваемых документов

Значение String.

## Синтаксис

*object***.TOC** [= *значение*]

Свойство TOC состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| значение | Значение String. |
