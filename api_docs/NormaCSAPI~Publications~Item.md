# Item (Publications)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Publication](NormaCSAPI~Publication.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Publications](NormaCSAPI~Publications.md). |
| Index | Обязательный. Значение Variant. |
