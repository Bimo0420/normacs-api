# ShowModal2 (About)

## Описание

Показывает модальный диалог с информацией о конфигурации NormaCS и подключенных модулях. В отличие от ShowModal, позволяет указать дескриптор родительского окна.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ShowModal2** [*HwndParent*]

Метод ShowModal2 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [About](NormaCSAPI_Dialogs~About.md). |
| HwndParent | Необязательный. Значение Variant. |
