# RevisionEditor Объект

## Описание

Объект RevisionEditor

## Объектная модель

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [CreateFutureRevision](NormaCSAPI~RevisionEditor~CreateFutureRevision.md) | Создает будущую (предстоящую) редакцию. Такая редакция в дальнейшем может быть сделана основной. |
| [CreateMainRevision](NormaCSAPI~RevisionEditor~CreateMainRevision.md) | Создает основную (действующую) редакцию. |
| [CreatePastRevision](NormaCSAPI~RevisionEditor~CreatePastRevision.md) | Создает прошлую (архивную) редакцию. Такая редакция не может быть сделана основной. |
| [RemoveRevision](NormaCSAPI~RevisionEditor~RemoveRevision.md) | Удаляет редакцию. Редакция не должна быть основной. |
| [TurnFutureRevisionIntoMainRevision](NormaCSAPI~RevisionEditor~TurnFutureRevisionIntoMainRevision.md) | Делает указанную будущую редакцию основной. Прежняя основная редакция сохраняется как прошлая. |
| [UpdateRevision](NormaCSAPI~RevisionEditor~UpdateRevision.md) | Изменяет атрибуты редакции. |
