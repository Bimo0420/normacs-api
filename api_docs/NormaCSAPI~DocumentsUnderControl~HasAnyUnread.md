# HasAnyUnread

## Описание

Имеется ли хотя бы один непрочтенный документ на контроле

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.HasAnyUnread**

Свойство HasAnyUnread состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [DocumentsUnderControl](NormaCSAPI~DocumentsUnderControl.md). |
