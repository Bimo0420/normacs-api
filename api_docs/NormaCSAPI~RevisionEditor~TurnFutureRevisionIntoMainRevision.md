# TurnFutureRevisionIntoMainRevision

## Описание

Делает указанную будущую редакцию основной. Прежняя основная редакция сохраняется как прошлая.

## Возвращаемое значение

Объект [Revision](NormaCSAPI~Revision.md).

## Синтаксис

```
object.TurnFutureRevisionIntoMainRevision (FutureRevision)
```

Метод TurnFutureRevisionIntoMainRevision состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [RevisionEditor](NormaCSAPI~RevisionEditor.md). |
| FutureRevision | Обязательный. Значение Variant. Сслылка на редакцию, или URL редакции. |
