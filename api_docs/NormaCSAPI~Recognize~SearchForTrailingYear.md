# SearchForTrailingYear

## Описание

При поиске документов учитывать цифры года в конце распознанного обозначения. В этом случае, при поиске документов для распознанного обозначения, наиболее подходящим кандидатом будет считаться документ с указанным годом. По умолчанию не установлено.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.SearchForTrailingYear** [= *значение*]

Свойство SearchForTrailingYear состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение Boolean. |
