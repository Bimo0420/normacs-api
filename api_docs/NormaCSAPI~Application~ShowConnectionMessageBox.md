# ShowConnectionMessageBox

## Описание

Разрешает отображать всплывающие сообщения о подключении к NormaCS. Deprecated, используйте вместо этого метода свойство [ShowConnectionMessages](NormaCSAPI~Application~ShowConnectionMessages.md) .

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ShowConnectionMessageBox**

Метод ShowConnectionMessageBox состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
