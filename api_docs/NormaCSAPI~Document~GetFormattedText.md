# GetFormattedText (Document)

## Описание

Формирует текст по указанному формату из атрибутов документа

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetFormattedText (Format)
```

Метод GetFormattedText состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
| Format | Обязательный. Значение String. |
