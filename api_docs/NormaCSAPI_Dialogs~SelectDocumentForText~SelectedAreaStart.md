# SelectedAreaStart

## Описание

Начальная позиция выделенной части текста начиная с единицы или ноль для всего текста

## Тип свойства

Значение Integer.

## Синтаксис

*object***.SelectedAreaStart**

Свойство SelectedAreaStart состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SelectDocumentForText](NormaCSAPI_Dialogs~SelectDocumentForText.md). |
