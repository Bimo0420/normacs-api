# StartDocumentSearchByTitle

## Описание

Активирует NormaCS и запускает поиск документов с указанным наименованием

## Возвращаемое значение

Нет.

## Синтаксис

*object***.StartDocumentSearchByTitle** *DocumentTitlePattern*

Метод StartDocumentSearchByTitle состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| DocumentTitlePattern | Обязательный. Значение String. |
