# Item (Developers)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Developer](NormaCSAPI~Developer.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Developers](NormaCSAPI~Developers.md). |
| Index | Обязательный. Значение Variant. |
