# RemoveStatus

## Описание

Удаляет указанный статус

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveStatus** *Status*

Метод RemoveStatus состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Status | Обязательный. Значение Variant. |
