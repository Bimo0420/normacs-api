# Item (Revisions)

## Описание

Возвращает элемент коллекции по указанному индексу

## Тип свойства

Объект [Revision](NormaCSAPI~Revision.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Revisions](NormaCSAPI~Revisions.md). |
| Index | Обязательный. Значение Variant. |
