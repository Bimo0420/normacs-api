# AddDocument

## Описание

Создает документ с указанными реквизитами

## Возвращаемое значение

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

```
object.AddDocument ([Index], [Number], [Name], [Id])
```

Метод AddDocument состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Index | Необязательный. Значение Variant. |
| Number | Необязательный. Значение Variant. |
| Name | Необязательный. Значение Variant. |
| Id | Необязательный. Значение Variant. |
