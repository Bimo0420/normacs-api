# ReadForCurrentUser

## Описание

Заполняет коллекцию из хранилища для текущего пользователя

## Возвращаемое значение

Нет.

## Синтаксис

*object***.ReadForCurrentUser**

Метод ReadForCurrentUser состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedActions](NormaCSAPI~UserDefinedActions.md). |
