# BuiltinIndex (Recognize2Pattern)

## Описание

Индекс шаблона в списке встроенных шаблонов, или ноль, если используется custom-шаблон.

## Тип свойства

Значение Integer.

## Синтаксис

*object***.BuiltinIndex** [= *значение*]

Свойство BuiltinIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Pattern](NormaCSAPI~Recognize2Pattern.md). |
| значение | Значение Integer. |
