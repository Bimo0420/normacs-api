# FindDocumentsWithSameYear

## Описание

Искать документ с обозначеним, имеющим цифры года, соответствующие распознанному обозначению. По умолчанию не установлено. Deprecated, используйте [Selectivity](NormaCSAPI~Recognize2Settings~Selectivity.md) .

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.FindDocumentsWithSameYear** [= *значение*]

Свойство FindDocumentsWithSameYear состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
| значение | Значение Boolean. |
