## Методы и свойства

Список разделов разыскиваемых документов, разделенных символом точки с запятой:Список_разделов = Раздел[;Раздел[;Раздел...]]

Раздел в списке задается указанием домена и классификатора и, необязательно, папки раздела:Раздел = Домен/Классификатор[/Папка[/Папка...]]

Домен указывается полностью.

Классификатор и имена папок можно указывать не полностью, если использовать символы шаблонов: звездочка для замены произвольного числа символов и знак вопроса для замены одного символа.

Примеры указания разделовnormacs.ru/Классификатор ISO/11 ЗДРАВООХРАНЕНИЕ/11.220 ВЕТЕРИНАРИЯnormacs.ru/Класс*/11*
