# NormaLink

## Описание

Возвращает ссылку на локальный файл C:/doc.normalink, используемую для формирования ссылок на документы NormaCS в среде AutoDesk AutoCAD.

## Тип свойства

Значение String.

## Синтаксис

*object***.NormaLink**

Свойство NormaLink состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
