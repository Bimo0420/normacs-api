# Length

## Описание

Длина поля, если требуется

## Тип свойства

Значение Integer.

## Синтаксис

*object***.Length** [= *значение*]

Свойство Length состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserFieldDef](NormaCSAPI~UserFieldDef.md). |
| значение | Значение Integer. |
