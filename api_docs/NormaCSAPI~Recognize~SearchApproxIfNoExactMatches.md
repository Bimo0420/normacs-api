# SearchApproxIfNoExactMatches

## Описание

Если не установлен флаг точного поиска (SearchExact), и при поиске не нашлись документы с обозначениями соответствующими распознанному, то искать приблизительно соответствующие. По умолчанию установлено.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.SearchApproxIfNoExactMatches** [= *значение*]

Свойство SearchApproxIfNoExactMatches состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение Boolean. |
