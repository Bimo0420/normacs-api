# CellMode

## Описание

Режим ячейки. В этом случае к распознанному обозначению автоматическм добавляется оставшийся текст.

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.CellMode** [= *значение*]

Свойство CellMode состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение Boolean. |
