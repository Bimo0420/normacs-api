# NormaCSDateSpanMeasureType

## Описание

Единица временного диапазона, заданного как количество месяцев или дней до даты последнего обновления домена.

## Значения

| Константа | Значение | Описание |
| --- | --- | --- |
|  | ncsDateSpanMeasureTypeMonth | 0 | Месяц |
|  | ncsDateSpanMeasureTypeDay | 1 | День |
