# Add (Documents)

## Описание

Добавляет указанный элемент (документ) в коллекцию

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Add** *pVal*

Метод Add состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Documents](NormaCSAPI~Documents.md). |
| pVal | Обязательный. Объект [Document](NormaCSAPI~Document.md). |
