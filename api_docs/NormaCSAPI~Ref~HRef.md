# HRef

## Описание

Символьная ссылка на связанный документ. Используется в ситуации, когда связанный документ не принадлежит к подключенным к домену базам.

## Тип свойства

Значение String.

## Синтаксис

*object***.HRef** [= *значение*]

Свойство HRef состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Ref](NormaCSAPI~Ref.md). |
| значение | Значение String. |
