# HasActive (Recognize2Patterns)

## Описание

Имеется ли в списке шаблонов хотя бы один активный шаблон

## Возвращаемое значение

Значение Boolean.

## Синтаксис

*object***.HasActive**

Метод HasActive состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Patterns](NormaCSAPI~Recognize2Patterns.md). |
