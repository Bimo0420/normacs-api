# Has (LinkFormats)

## Описание

Имеется ли в списке формат с указанным именем или ссылкой

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.Has (NameOrRef)
```

Метод Has состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [LinkFormats](NormaCSAPI~LinkFormats.md). |
| NameOrRef | Обязательный. Значение Variant. |
