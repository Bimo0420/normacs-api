# Clone (Recognize2Pattern)

## Описание

Создает копию текущего шаблона (не включенную в список шаблонов)

## Возвращаемое значение

Объект [Recognize2Pattern](NormaCSAPI~Recognize2Pattern.md).

## Синтаксис

*object***.Clone**

Метод Clone состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Pattern](NormaCSAPI~Recognize2Pattern.md). |
