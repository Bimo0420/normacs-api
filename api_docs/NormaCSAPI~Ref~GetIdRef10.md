# GetIdRef10

## Описание

Возвращает идентификатор связанного документа в десятичном формате.

## Возвращаемое значение

Значение String.

## Синтаксис

*object***.GetIdRef10**

Метод GetIdRef10 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Ref](NormaCSAPI~Ref.md). |
