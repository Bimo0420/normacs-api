# Remove (LinkFormats)

## Описание

Удаляет формат

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Remove** *Index*

Метод Remove состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [LinkFormats](NormaCSAPI~LinkFormats.md). |
| Index | Обязательный. Значение Variant. |
