# StartDocumentSearchByNumber

## Описание

Активирует NormaCS и запускает поиск документов по указанному номером

## Возвращаемое значение

Нет.

## Синтаксис

*object***.StartDocumentSearchByNumber** *DocumentNumberPattern*

Метод StartDocumentSearchByNumber состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| DocumentNumberPattern | Обязательный. Значение String. |
