# MatchedDocumentIndex (Recognize)

## Описание

Порядковый номер в списке документа, обозначение которого совпадает с распознанным с точностью до разделителей

## Тип свойства

Значение Integer.

## Синтаксис

*object***.MatchedDocumentIndex**

Свойство MatchedDocumentIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
