# UnambiguousIndexes

## Описание

Возвращает список шаблонов 'достоверных' индексов, то есть таких, для которых не может быть более одного документа с одним и тем же обозначением

## Тип свойства

Объект [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md).

## Синтаксис

*object***.UnambiguousIndexes**

Свойство UnambiguousIndexes состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Settings](NormaCSAPI~Recognize2Settings.md). |
