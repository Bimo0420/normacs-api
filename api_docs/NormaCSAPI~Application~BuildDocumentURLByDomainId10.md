# BuildDocumentURLByDomainId10

## Описание

Конструирует адрес документа по указанному домену и идентификатору в десятичном формате

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.BuildDocumentURLByDomainId10 (Domain, Id10)
```

Метод BuildDocumentURLByDomainId10 состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| Domain | Обязательный. Значение String. |
| Id10 | Обязательный. Значение String. |
