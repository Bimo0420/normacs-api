# MarkedAreaEnd

## Описание

Конечная позиция подкрашенной части текста начиная с единицы

## Тип свойства

Значение Integer.

## Синтаксис

*object***.MarkedAreaEnd** [= *значение*]

Свойство MarkedAreaEnd состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SelectDocumentForText](NormaCSAPI_Dialogs~SelectDocumentForText.md). |
| значение | Значение Integer. |
