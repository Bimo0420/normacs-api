# SetParametersBySearchString

## Описание

Устанавливает параметры поиска на основании анализа поисковой строки

## Возвращаемое значение

Нет.

## Синтаксис

*object***.SetParametersBySearchString** *SearchString*

Метод SetParametersBySearchString состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| SearchString | Обязательный. Значение String. |
