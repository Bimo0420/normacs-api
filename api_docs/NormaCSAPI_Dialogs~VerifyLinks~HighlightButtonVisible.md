# HighlightButtonVisible

## Описание

Видимость кнопок выделения и снятия выделения проблемных документов

## Тип свойства

Значение Boolean.

## Синтаксис

*object***.HighlightButtonVisible** [= *значение*]

Свойство HighlightButtonVisible состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [VerifyLinks](NormaCSAPI_Dialogs~VerifyLinks.md). |
| значение | Значение Boolean. |
