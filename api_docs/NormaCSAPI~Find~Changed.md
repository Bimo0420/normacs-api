# Changed

## Описание

Время изменения разыскиваемых документов

## Тип свойства

Объект [DateSpan](NormaCSAPI~DateSpan.md).

## Синтаксис

**[Set]** *object***.Changed** [= *DateSpan*]

Свойство Changed состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| DateSpan | Объект [DateSpan](NormaCSAPI~DateSpan.md). |
