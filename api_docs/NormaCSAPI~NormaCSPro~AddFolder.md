# AddFolder

## Описание

Создает корневую или доченрнюю папку с указанным названием и параметром упорядочивания

## Возвращаемое значение

Объект [Folder](NormaCSAPI~Folder.md).

## Синтаксис

```
object.AddFolder (Parent, Name, [Sort], [Id])
```

Метод AddFolder состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Parent | Обязательный. Значение Variant. |
| Name | Обязательный. Значение String. |
| Sort | Необязательный. Значение Variant. |
| Id | Необязательный. Значение Variant. |
