# Clone (UnambiguousIndex)

## Описание

Создает копию текущего шаблона (не включенную в список шаблонов)

## Возвращаемое значение

Объект [UnambiguousIndex](NormaCSAPI~UnambiguousIndex.md).

## Синтаксис

*object***.Clone**

Метод Clone состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndex](NormaCSAPI~UnambiguousIndex.md). |
