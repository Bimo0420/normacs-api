# Launch

## Описание

Запускает приложение NormaCS, если еще не запущено

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Launch**

Метод Launch состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
