# IconIndex (UserDefinedAction)

## Описание

Индекс изображения элемента в исполняемом файле или в .ico файле

## Тип свойства

Значение Integer.

## Синтаксис

*object***.IconIndex** [= *значение*]

Свойство IconIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UserDefinedAction](NormaCSAPI~UserDefinedAction.md). |
| значение | Значение Integer. |
