# Appendix Объект

## Описание

Описывает отдельное приложение к документу.Список имеющихся в документе приложений возвращает свойство документа [Document.Appendixes](NormaCSAPI~Document~Appendixes.md).Используйте методы редактора NormaCS Pro для выгрузки, добавления, изменения или удаления приложений.
