# RegExp (Recognize)

## Описание

Текст, задающий perl-совместимое регулярное выражения для распознавания обозначения. Синтаксис см. http://www.pcre.org/pcre.txt

## Тип свойства

Значение String.

## Синтаксис

*object***.RegExp** [= *значение*]

Свойство RegExp состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| значение | Значение String. |

## Замечания

Регулярное выражение обязательно должно иметь ровно две группы захвата (выделяются круглыми скобками) - для запоминания распознанных индекса и номера.
