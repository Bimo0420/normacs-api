# AddDocumentReplacedBy

## Описание

Добавляет указанную замену (объект Replacement или Document, или URL). Вызовите SaveDocument для сохранения в базе измененных атрибутов документа.

## Возвращаемое значение

Объект [Replacement](NormaCSAPI~Replacement.md).

## Синтаксис

```
object.AddDocumentReplacedBy (pDocument, [Replacement])
```

Метод AddDocumentReplacedBy состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| pDocument | Обязательный. Объект [Document](NormaCSAPI~Document.md). |
| Replacement | Необязательный. Значение Variant. |
