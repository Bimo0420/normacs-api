# SelectedDocumentIndex

## Описание

Индекс выбранного документа начиная с единицы или ноль если документ не выбран

## Тип свойства

Значение Integer.

## Синтаксис

*object***.SelectedDocumentIndex** [= *значение*]

Свойство SelectedDocumentIndex состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [SelectDocumentForText](NormaCSAPI_Dialogs~SelectDocumentForText.md). |
| значение | Значение Integer. |
