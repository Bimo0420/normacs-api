# CancelWait

## Описание

Посылает сигнал к завершению цикла ожидания события, вызванного функциями WaitForEvent или WaitForEventEx

## Возвращаемое значение

Нет.

## Синтаксис

*object***.CancelWait**

Метод CancelWait состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
