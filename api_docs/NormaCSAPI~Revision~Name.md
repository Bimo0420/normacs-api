# Name (Revision)

## Описание

Наименование редакции

## Тип свойства

Значение String.

## Синтаксис

*object***.Name**

Свойство Name состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Revision](NormaCSAPI~Revision.md). |

## Замечания

Этот атрибут нельзя редактировать, он обновляется автоматически при изменении типа редакции. В списке редакций приложения NormaCS Pro этому атрибуту соответствует поле "Название".
