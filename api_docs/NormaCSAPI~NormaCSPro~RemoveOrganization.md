# RemoveOrganization

## Описание

Удаляет указанную организацию из справочника организаций домена

## Возвращаемое значение

Нет.

## Синтаксис

*object***.RemoveOrganization** *Organization*

Метод RemoveOrganization состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [NormaCSPro](NormaCSAPI~NormaCSPro.md). |
| Organization | Обязательный. Значение Variant. |
