# Sort

## Описание

Число, определяющее относительное положение папки в списке папок родителя

## Тип свойства

Значение Long.

## Синтаксис

*object***.Sort** [= *значение*]

Свойство Sort состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Folder](NormaCSAPI~Folder.md). |
| значение | Значение Long. |
