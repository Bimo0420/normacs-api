# ShowModal (FindSettings)

## Описание

Показывает модальный диалог настроек поиска

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.ShowModal (AddinSettings, [HwndParent])
```

Метод ShowModal состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [FindSettings](NormaCSAPI_Dialogs~FindSettings.md). |
| AddinSettings | Обязательный. Объект Object. |
| HwndParent | Необязательный. Значение Variant. |
