# Count (WantRefinementIndexes)

## Описание

Возвращает количество элементов в списке шаблонов

## Тип свойства

Значение Long.

## Синтаксис

*object***.Count**

Свойство Count состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [WantRefinementIndexes](NormaCSAPI~WantRefinementIndexes.md). |
