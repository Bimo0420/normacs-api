# Has (UnambiguousIndexes)

## Описание

Имеется ли в списке шаблонов шаблон с указанным именем или built-in индексом

## Возвращаемое значение

Значение Boolean.

## Синтаксис

```
object.Has (BuiltinIndexOrNameOrRef)
```

Метод Has состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [UnambiguousIndexes](NormaCSAPI~UnambiguousIndexes.md). |
| BuiltinIndexOrNameOrRef | Обязательный. Значение Variant. |
