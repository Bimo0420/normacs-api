# Item (Recognize2Patterns)

## Описание

Возвращает шаблон распознавания по указанному индексу, имени или ссылке

## Тип свойства

Объект [Recognize2Pattern](NormaCSAPI~Recognize2Pattern.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Patterns](NormaCSAPI~Recognize2Patterns.md). |
| Index | Обязательный. Значение Variant. |
