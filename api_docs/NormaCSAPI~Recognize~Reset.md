# Reset (Recognize)

## Описание

Сбрасывает выходные параметры, в том числе, освобождает коллекцию документов для распознанного обозначения. Если указано, очищает кеш распознанных документов.

## Возвращаемое значение

Нет.

## Синтаксис

*object***.Reset** [*clearDocumentCache*]

Метод Reset состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize](NormaCSAPI~Recognize.md). |
| clearDocumentCache | Необязательный. Значение Variant. |
