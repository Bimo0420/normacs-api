# Domain (Document)

## Описание

Возвращает имя домена, к которому принадлежит данный документ. Вместе с идентификатором однозначно определяет документ в системе NormaCS

## Тип свойства

Значение String.

## Синтаксис

*object***.Domain**

Свойство Domain состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
