## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [Execute](NormaCSAPI~Recognize2~Execute.md) | Выполняет распознавание обозначения документа в тексте. При успешном распознавании обновляет NextPos, и, если указано, формирует список документов, подходящих к распознанному обозначению. |
| [Reset](NormaCSAPI~Recognize2~Reset.md) | Сбрасывает выходные параметры, в том числе, освобождает коллекцию документов для распознанного обозначения. Если указано, очищает кеш распознанных документов. |
