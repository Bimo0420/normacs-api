# Position (Recognize2Pattern)

Позиция шаблона в списке шаблонов, начиная с единицы

## Тип свойства

Значение Integer.

## Синтаксис

*object***.Position** [= *значение*]

Свойство Position состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Pattern](NormaCSAPI~Recognize2Pattern.md). |
| значение | Значение Integer. |
