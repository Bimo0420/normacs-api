# SetlinkLogger Объект

## Описание

Ведение протокола расстановки ссылок

## Методы и свойства

| Методы |  | Методы |
| --- | --- | --- |
|  | Методы |
| [GetLastSessionId](NormaCSAPI~SetlinkLogger~GetLastSessionId.md) | Возвращает идентификатор последней сессии расстановки ссылок |
| [GetLogAsHtml](NormaCSAPI~SetlinkLogger~GetLogAsHtml.md) | Возвращает html-представление протокола указанной сессии расстановки ссылок |
| [GetLogAsXml](NormaCSAPI~SetlinkLogger~GetLogAsXml.md) | Возвращает xml-представление протокола указанной сессии расстановки ссылок |
| [GetPath](NormaCSAPI~SetlinkLogger~GetPath.md) | Возвращает путь на документ для указанной сессии расстановки ссылок |
| [GetSummaryAsXml](NormaCSAPI~SetlinkLogger~GetSummaryAsXml.md) | Возвращает xml-представление сводной информации для указанной сессии расстановки ссылок |
| [HasLog](NormaCSAPI~SetlinkLogger~HasLog.md) | Возвращает True если в базе имеются данные о расстановке ссылок с указанным тэгом |
| [IsAnalysisOnly](NormaCSAPI~SetlinkLogger~IsAnalysisOnly.md) | Указанная сессия предназначена только для анализа документа |
| [LogOperation](NormaCSAPI~SetlinkLogger~LogOperation.md) | Записывает в протокол расстановки ссылок |
| [StartSession](NormaCSAPI~SetlinkLogger~StartSession.md) | Начинает сессию расстановки ссылок. Возвращает идентификатор сессии. |
| [StartSessionForAnalysis](NormaCSAPI~SetlinkLogger~StartSessionForAnalysis.md) | Начинает сессию анализа ссылок. Возвращает идентификатор сессии. |
