# AssignTo (Recognize2Patterns)

## Описание

Копирует список шаблонов в указанный

## Возвращаемое значение

Нет.

## Синтаксис

*object***.AssignTo** *pVal*

Метод AssignTo состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Recognize2Patterns](NormaCSAPI~Recognize2Patterns.md). |
| pVal | Обязательный. Объект [Recognize2Patterns](NormaCSAPI~Recognize2Patterns.md). |
