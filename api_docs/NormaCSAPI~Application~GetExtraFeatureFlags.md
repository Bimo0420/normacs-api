# GetExtraFeatureFlags

## Описание

Возвращает строку флагов feature с указанным именем

## Возвращаемое значение

Значение String.

## Синтаксис

```
object.GetExtraFeatureFlags (FeatureName)
```

Метод GetExtraFeatureFlags состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| FeatureName | Обязательный. Значение String. |
