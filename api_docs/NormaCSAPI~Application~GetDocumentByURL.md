# GetDocumentByURL

## Описание

Возвращает документ с указанным адресом

## Возвращаемое значение

Объект [Document](NormaCSAPI~Document.md).

## Синтаксис

```
object.GetDocumentByURL (URL)
```

Метод GetDocumentByURL состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Application](NormaCSAPI~Application.md). |
| URL | Обязательный. Значение String. |
