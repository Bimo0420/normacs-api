# Item (Domains)

## Описание

Возвращает элемент коллекции (домен) по указанному индексу

## Тип свойства

Объект [Domain](NormaCSAPI~Domain.md).

## Синтаксис

```
object.Item(Index)
```

Свойство Item состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Domains](NormaCSAPI~Domains.md). |
| Index | Обязательный. Значение Variant. |
