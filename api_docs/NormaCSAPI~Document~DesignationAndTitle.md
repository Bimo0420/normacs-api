# DesignationAndTitle

## Описание

Возвращает обозначение и наименование документа одной строкой.

## Тип свойства

Значение String.

## Синтаксис

*object***.DesignationAndTitle**

Свойство DesignationAndTitle состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Document](NormaCSAPI~Document.md). |
