# Limit

## Описание

Максимальное количество возвращаемых документов, по умолчанию 500. Значение ноль означает не ограничивать выдачу поиска (не рекомендуется).

## Тип свойства

Значение Integer.

## Синтаксис

*object***.Limit** [= *значение*]

Свойство Limit состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [Find](NormaCSAPI~Find.md). |
| значение | Значение Integer. |
