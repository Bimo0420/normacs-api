# AddLink

## Описание

Добавляет ссылку для проверки

## Возвращаемое значение

Нет.

## Синтаксис

*object***.AddLink** *Url*, *Text*, [*LinkInTextStartPos*], [*LinkInTextEndPos*], [*LinkInSourceDocumentPageNumber*], [*TextInSourceDocumentLocation*]

Метод AddLink состоит из:

| Часть | Описание |
| --- | --- |
| объект | Объект типа [VerifyLinks](NormaCSAPI_Dialogs~VerifyLinks.md). |
| Url | Обязательный. Значение String. |
| Text | Обязательный. Значение String. |
| LinkInTextStartPos | Необязательный. Значение Variant. |
| LinkInTextEndPos | Необязательный. Значение Variant. |
| LinkInSourceDocumentPageNumber | Необязательный. Значение Variant. |
| TextInSourceDocumentLocation | Необязательный. Значение Variant. |
